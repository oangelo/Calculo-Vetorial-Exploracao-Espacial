• EXERCÍCIO CONIC-5 [NÍVEL ALFA]
    ORIGEM: Estudo de Trajetórias de Reentrada [PROGRAMA GEMINI, 1964]

    Assim como fizemos com a elipse, vamos extrair informações de uma parábola a partir de sua equação. Interceptamos:

    \[y = \frac{1}{8}x^2\]

    Tarefa:
    (a) Reescreva na forma \(x^2 = 4py\). Qual é o valor de \(4p\)?
    (b) Determine \(p\). Onde está o foco?
    (c) Onde está a diretriz?
    (d) O vértice está na origem. A parábola abre para cima ou para baixo? Justifique.

    ARQUIVO AUXILIAR [SIGMA-5]
    [NOTA TÉCNICA]: A forma \(y = \frac{1}{8}x^2\) equivale a \(x^2 = 8y\). Compare com \(x^2 = 4py\) para identificar \(4p = 8\). O foco de uma parábola vertical \(x^2 = 4py\) está em \((0, p)\).
    RELATÓRIO DE CÁLCULO [SIGMA-5 REQUERIDO]

    (a) Reescrevendo:
    \[y = \frac{1}{8}x^2 \implies x^2 = 8y\]
    Comparando com \(x^2 = 4py\): \(4p = 8\).

    (b) Valor de \(p\) e foco:
    \[4p = 8 \implies p = 2\]
    Foco: \(F(0, 2)\).

    (c) Diretriz: \(y = -p = -2\).

    (d) Orientação: Como o coeficiente de \(x^2\) é positivo (\(\frac{1}{8} > 0\)), a parábola abre para cima. O vértice é o ponto mais baixo da curva em \(V(0, 0)\).
