• EXERCÍCIO CONIC-6 [NÍVEL BETA]
    ORIGEM: Agência [REDACTED] - Seção Órbitas Hiperbólicas

    Detectamos uma trajetória de passagem (flyby) de um objeto não identificado. A equação interceptada é:

    \[\frac{x^2}{9} - \frac{y^2}{16} = 1\]

    Tarefa:
    (a) Identifique o tipo de cônica. O sinal de subtração entre os termos quadráticos indica o quê?
    (b) Faça \(y = 0\). O que os valores de \(x\) representam?
    (c) Determine \(a\), \(b\) e calcule \(c\).
    (d) Calcule as equações das assíntotas. (Dica: para \(|x|\) grande, o que acontece com \(y\)? A curva se aproxima de quais retas?)
    (e) Calcule a excentricidade. Ela é maior ou menor que 1? Por quê?

    ARQUIVO AUXILIAR [SIGMA-6]
    [NOTA TÉCNICA]: Para hipérboles, \(c^2 = a^2 + b^2\) (diferente da elipse onde é \(c^2 = a^2 - b^2\)). As assíntotas são \(y = \pm\frac{b}{a}x\).
    RELATÓRIO DE CÁLCULO [SIGMA-6 REQUERIDO]

    (a) Identificação: Sinal de subtração → hipérbole. O termo positivo é \(x^2\), logo o eixo transverso é horizontal.

    (b) Fazendo \(y = 0\):
    \[\frac{x^2}{9} = 1 \implies x = \pm 3\]
    Os pontos \((-3, 0)\) e \((3, 0)\) são os vértices.

    (c) Parâmetros:
    \(a = 3\), \(b = 4\), \(c^2 = 9 + 16 = 25 \implies c = 5\).

    (d) Assíntotas:
    \[y = \pm\frac{b}{a}x = \pm\frac{4}{3}x\]

    (e) Excentricidade:
    \[e = \frac{c}{a} = \frac{5}{3} \approx 1{,}667 > 1\]
    Hipérboles sempre têm \(e > 1\), pois \(c > a\) (ao contrário da elipse onde \(c < a\)).
