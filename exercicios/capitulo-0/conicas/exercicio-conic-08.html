• EXERCÍCIO CONIC-08 [NÍVEL BETA]
    ORIGEM: Classificação de Trajetórias - Centro de Análise Orbital [SIGMA-2]

    Classifique cada equação abaixo como círculo, elipse, parábola ou hipérbole usando apenas o discriminante \(B^2 - 4AC\) da forma geral \(Ax^2 + Bxy + Cy^2 + Dx + Ey + F = 0\):

    (a) \(4x^2 + y^2 - 8x + 4y + 4 = 0\)
    (b) \(x^2 - 4y^2 + 6x + 8y - 11 = 0\)
    (c) \(9x^2 + 6xy + y^2 - 4 = 0\)
    (d) \(x^2 + y^2 - 6x + 2y + 1 = 0\)

    ARQUIVO AUXILIAR [SIGMA-2]
    [NOTA TÉCNICA]: Se \(B^2 - 4AC < 0\) → elipse (ou círculo se \(A=C, B=0\)). Se \(B^2 - 4AC = 0\) → parábola. Se \(B^2 - 4AC > 0\) → hipérbole. Identifique \(A\), \(B\), \(C\) de cada equação.
    RELATÓRIO DE CÁLCULO [SIGMA-2 REQUERIDO]

    (a) \(A=4, B=0, C=1\). Discriminante: \(0 - 16 = -16 < 0\). Como \(A \neq C\): elipse.

    (b) \(A=1, B=0, C=-4\). Discriminante: \(0 - (-16) = 16 > 0\): hipérbole.

    (c) \(A=9, B=6, C=1\). Discriminante: \(36 - 36 = 0\): parábola.

    (d) \(A=1, B=0, C=1\). Discriminante: \(0 - 4 = -4 < 0\). Como \(A=C\) e \(B=0\): círculo.
