• EXERCÍCIO CONIC-14 [NÍVEL GAMMA]
    ORIGEM: Classificação de Trajetórias Codificadas - Projeto Interceptação [SIGMA-3]

    Mais dois casos para treinar a classificação por discriminante:

    (a) \(x^2 + 4y^2 - 6x + 16y + 21 = 0\)
    (b) \(2x^2 - 3y^2 + 8x + 12y - 4 = 0\)

    Para cada uma:
    (i) Classifique pelo discriminante \(B^2 - 4AC\).
    (ii) Complete os quadrados para obter a forma canônica.
    (iii) Determine centro, eixos e focos (quando aplicável).

    ARQUIVO AUXILIAR [SIGMA-14]
    [NOTA TÉCNICA]: Use o discriminante para classificar primeiro. Depois complete os quadrados para obter a forma canônica.
    RELATÓRIO DE CÁLCULO [SIGMA-14 REQUERIDO]

    (a) Discriminante: \(0 - 4(1)(4) = -16 < 0\) → elipse.
    \[(x-3)^2 + 4(y+2)^2 = 4\]
    \[\frac{(x-3)^2}{4} + (y+2)^2 = 1\]
    Centro \((3, -2)\), \(a = 2, b = 1\), \(c = \sqrt{3}\).

    (b) Discriminante: \(0 - 4(2)(-3) = 24 > 0\) → hipérbole.
    \[\frac{(x+2)^2}{2} - \frac{(y-2)^2}{4/3} = 1\]
    Centro \((-2, 2)\), \(a = \sqrt{2}, b = \frac{2}{\sqrt{3}}\).
