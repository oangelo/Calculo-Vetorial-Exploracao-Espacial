• EXERCÍCIO CONIC-14 [NÍVEL GAMMA]
    ORIGEM: Arquivos Secretos [NÍVEL GAMMA - DADOS DESCRIPTOGRAFADOS]

    Já sabemos construir cônicas pela definição, extrair informações de equações canônicas e completar quadrados. Mas e quando a equação vem na forma geral, sem sabermos de antemão se é elipse, parábola ou hipérbole? O discriminante \(\Delta = B^2 - 4AC\) permite classificar sem resolver nada.

    Classifique as seguintes equações usando o discriminante e, em seguida, confirme completando os quadrados:

    Trajetória Alfa: \(4x^2 + 9y^2 - 16x + 18y - 11 = 0\)
    Trajetória Beta: \(y^2 - 8x + 6y + 1 = 0\)

    Tarefa para cada trajetória:
    (a) Identifique \(A\), \(B\), \(C\) e calcule \(\Delta = B^2 - 4AC\).
    (b) Classifique: \(\Delta < 0\) → elipse, \(\Delta = 0\) → parábola, \(\Delta > 0\) → hipérbole.
    (c) Complete os quadrados e obtenha a forma canônica. Confirme a classificação.

    ARQUIVO AUXILIAR [SIGMA-6]
    [NOTA TÉCNICA]: Na equação geral \(Ax^2 + Bxy + Cy^2 + Dx + Ey + F = 0\), identifique os coeficientes dos termos quadráticos. Para Trajetória Alfa: \(A = 4, B = 0, C = 9\).
    RELATÓRIO DE CÁLCULO [SIGMA-6 REQUERIDO]

    === TRAJETÓRIA ALFA ===

    (a) Discriminante: \(A = 4, B = 0, C = 9\)
    \[\Delta = 0 - 4(4)(9) = -144 < 0 \implies \textbf{ELIPSE}\]

    (c) Completando quadrados:
    \[4(x^2 - 4x) + 9(y^2 + 2y) = 11\]
    \[4(x-2)^2 + 9(y+1)^2 = 36\]
    \[\frac{(x-2)^2}{9} + \frac{(y+1)^2}{4} = 1\]
    Centro \((2, -1)\), \(a = 3, b = 2\). Confirmado: elipse. ✓

    === TRAJETÓRIA BETA ===

    (a) Discriminante: \(A = 0, B = 0, C = 1\)
    \[\Delta = 0 - 4(0)(1) = 0 \implies \textbf{PARÁBOLA}\]

    (c) Completando quadrado:
    \[(y+3)^2 = 8(x+1)\]
    Vértice \((-1, -3)\), \(p = 2\). Confirmado: parábola. ✓
