• EXERCÍCIO CONIC-16 [NÍVEL GAMMA]
    ORIGEM: Classificação de Trajetórias Codificadas - Projeto Interceptação [SIGMA-3]

    Identifique o tipo de cônica e reescreva na forma canônica (completo os quadrados se necessário):

    (a) \(x^2 + 4y^2 - 6x + 16y + 21 = 0\)
    (b) \(2x^2 - 3y^2 + 8x + 12y - 4 = 0\)
    (c) \(y^2 - 10x + 6y + 29 = 0\)

    ARQUIVO AUXILIAR [SIGMA-3]
    [NOTA TÉCNICA]: Use o discriminante para classificar primeiro. Depois complete os quadrados para obter a forma canônica. Lembre-se: para parábolas, isole a variável que não está ao quadrado.
    RELATÓRIO DE CÁLCULO [SIGMA-3 REQUERIDO]

    (a) Discriminante: \(0 - 4(1)(4) = -16 < 0\) → elipse.
    \((x-3)^2 - 9 + 4(y+2)^2 - 16 + 21 = 0\).
    \((x-3)^2 + 4(y+2)^2 = 4\).
    \(\dfrac{(x-3)^2}{4} + (y+2)^2 = 1\). Centro \((3,-2)\), \(a=2\), \(b=1\).

    (b) Discriminante: \(0 - 4(2)(-3) = 24 > 0\) → hipérbole.
    \(2(x^2+4x) - 3(y^2-4y) = 4\).
    \(2[(x+2)^2-4] - 3[(y-2)^2-4] = 4\).
    \(2(x+2)^2 - 3(y-2)^2 = 4 + 8 - 12 = 0\).
    \(\dfrac{(x+2)^2}{2} - \dfrac{(y-2)^2}{4/3} = 1\). Centro \((-2,2)\).

    (c) Discriminante: \(0 - 0 = 0\) → parábola (sem \(x^2\)).
    \((y+3)^2 - 9 - 10x + 29 = 0\).
    \((y+3)^2 = 10x - 20 = 10(x-2)\).
    \((y+3)^2 = 10(x-2)\). Vértice \((2,-3)\), \(p = 5/2\), abre para a direita.
