• EXERCÍCIO CONIC-20 [NÍVEL OMEGA]
    ORIGEM: Verificação de Dados Interceptados - Centro de Análise [SIGMA-4]

    Interceptamos uma equação com termo misto \(xy\). Sem usar rotação de coordenadas, classifique a cônica e determine se representa uma curva real ou imaginária:

    \[ 3x^2 + 2xy + 3y^2 - 8 = 0 \]

    Tarefa:
    (a) Calcule o discriminante \(B^2 - 4AC\) e classifique o tipo de cônica.
    (b) Complete os quadrados agrupando termos. Dica: \(3x^2 + 2xy + 3y^2\) pode ser reescrito como soma de quadrados. Tente \(2(x^2 + y^2) + (x+y)^2\).
    (c) A equação descreve uma curva real? Se sim, determine os semi-eixos.

    RELATÓRIO DE CÁLCULO [SIGMA-4 REQUERIDO]

    (a) Discriminante: \(A = 3, B = 2, C = 3\). \[\Delta = 4 - 36 = -32 < 0 \implies \textbf{ELIPSE}\]

    (b) Completando quadrados sem rotação:
    Observe que \(3x^2 + 2xy + 3y^2 = 2(x^2 + y^2) + (x^2 + 2xy + y^2) = 2(x^2 + y^2) + (x+y)^2\).
    A equação fica: \(2(x^2 + y^2) + (x+y)^2 = 8\).
    Como \(x^2 + y^2 \geq 0\) e \((x+y)^2 \geq 0\), a curva existe (é uma elipse real, rotacionada de 45°).

    (c) Semi-eixos: Substituindo \(u = \frac{x+y}{\sqrt{2}}\) e \(v = \frac{x-y}{\sqrt{2}}\) (rotação de 45°), obtemos:
    \(4u^2 + 2v^2 = 8\), ou \(\frac{u^2}{2} + v^2 = 1\).
    Semi-eixo maior: \(a = \sqrt{2}\) na direção \(u\). Semi-eixo menor: \(b = 1\) na direção \(v\).
