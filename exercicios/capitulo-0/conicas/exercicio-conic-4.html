• EXERCÍCIO CONIC-4
    ORIGEM: Arquivos Secretos [NÍVEL OMEGA - DADOS DESCRIPTOGRAFADOS]

    (Atividade orientada) Interceptamos quatro equações de trajetórias codificadas em sua forma geral \(Ax^2 + Bxy + Cy^2 + Dx + Ey + F = 0\). Sua missão é classificar cada cônica e, quando possível, obter sua forma canônica com todos os parâmetros.

    Trajetórias interceptadas:

    Trajetória Alfa: \(4x^2 + 9y^2 - 16x + 18y - 11 = 0\)
    Trajetória Beta: \(x^2 - 4y^2 - 6x - 16y - 19 = 0\)
    Trajetória Gama: \(y^2 - 8x + 6y + 1 = 0\)
    Trajetória Delta: \(9x^2 + 4y^2 - 36 = 0\)

    Tarefa para cada trajetória:
    (a) Use o discriminante \(\Delta = B^2 - 4AC\) para classificar a cônica.
    (b) Complete quadrados e reescreva na forma canônica.
    (c) Determine os parâmetros fundamentais (\(a\), \(b\), \(c\), \(e\) quando aplicável).

    RELATÓRIO DE CÁLCULO

    === TRAJETÓRIA ALFA: \(4x^2 + 9y^2 - 16x + 18y - 11 = 0\) ===

    (a) Discriminante: \(A = 4,\ B = 0,\ C = 9\)
    \[\Delta = 0^2 - 4(4)(9) = -144 < 0 \implies \textbf{ELIPSE}\]

    (b) Completando quadrados:
    \[4(x^2 - 4x) + 9(y^2 + 2y) = 11\]
    \[4(x^2 - 4x + 4) + 9(y^2 + 2y + 1) = 11 + 16 + 9\]
    \[4(x - 2)^2 + 9(y + 1)^2 = 36\]
    \[\frac{(x - 2)^2}{9} + \frac{(y + 1)^2}{4} = 1\]

    (c) Parâmetros: Centro em \((2,\, -1)\)
    \(a^2 = 9 \implies a = 3\), \(b^2 = 4 \implies b = 2\)
    \(c^2 = 9 - 4 = 5 \implies c = \sqrt{5} \approx 2{,}236\)
    \(e = \frac{\sqrt{5}}{3} \approx 0{,}745\)

    === TRAJETÓRIA BETA: \(x^2 - 4y^2 - 6x - 16y - 19 = 0\) ===

    (a) Discriminante: \(A = 1,\ B = 0,\ C = -4\)
    \[\Delta = 0^2 - 4(1)(-4) = 16 > 0 \implies \textbf{HIPÉRBOLE}\]

    (b) Completando quadrados:
    \[(x^2 - 6x) - 4(y^2 + 4y) = 19\]
    \[(x^2 - 6x + 9) - 4(y^2 + 4y + 4) = 19 + 9 - 16\]
    \[(x - 3)^2 - 4(y + 2)^2 = 12\]
    \[\frac{(x - 3)^2}{12} - \frac{(y + 2)^2}{3} = 1\]

    (c) Parâmetros: Centro em \((3,\, -2)\)
    \(a^2 = 12 \implies a = 2\sqrt{3} \approx 3{,}464\)
    \(b^2 = 3 \implies b = \sqrt{3} \approx 1{,}732\)
    \(c^2 = 12 + 3 = 15 \implies c = \sqrt{15} \approx 3{,}873\)
    \(e = \frac{\sqrt{15}}{2\sqrt{3}} = \frac{\sqrt{5}}{2} \approx 1{,}118\)

    === TRAJETÓRIA GAMA: \(y^2 - 8x + 6y + 1 = 0\) ===

    (a) Discriminante: \(A = 0,\ B = 0,\ C = 1\)
    \[\Delta = 0^2 - 4(0)(1) = 0 \implies \textbf{PARÁBOLA}\]

    (b) Isolando e completando quadrado:
    \[y^2 + 6y = 8x - 1\]
    \[(y^2 + 6y + 9) = 8x - 1 + 9\]
    \[(y + 3)^2 = 8x + 8 = 8(x + 1)\]
    \[(y + 3)^2 = 8(x + 1)\]

    (c) Parâmetros: Vértice em \((-1,\, -3)\)
    Forma: \((y - k)^2 = 4p(x - h)\) com \(4p = 8 \implies p = 2\)
    Foco: \((-1 + 2,\, -3) = (1,\, -3)\)
    Diretriz: \(x = -1 - 2 = -3\)

    === TRAJETÓRIA DELTA: \(9x^2 + 4y^2 - 36 = 0\) ===

    (a) Discriminante: \(A = 9,\ B = 0,\ C = 4\)
    \[\Delta = 0^2 - 4(9)(4) = -144 < 0 \implies \textbf{ELIPSE}\]

    (b) Forma canônica:
    \[9x^2 + 4y^2 = 36\]
    \[\frac{x^2}{4} + \frac{y^2}{9} = 1\]

    (c) Parâmetros: Centro na origem
    \(a^2 = 9 \implies a = 3\), \(b^2 = 4 \implies b = 2\)
    Eixo maior ao longo do eixo \(y\) (pois \(9 > 4\))
    \(c^2 = 9 - 4 = 5 \implies c = \sqrt{5} \approx 2{,}236\)
    \(e = \frac{\sqrt{5}}{3} \approx 0{,}745\)
    Focos: \((0,\, -\sqrt{5})\) e \((0,\, \sqrt{5})\)
