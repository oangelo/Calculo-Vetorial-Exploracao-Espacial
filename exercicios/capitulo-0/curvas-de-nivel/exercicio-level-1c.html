• EXERCÍCIO LEVEL-1C [NÍVEL ALFA]
    ORIGEM: Mapeamento de Crateras - Missão Lunar Orbiter, 1966

    Você já viu círculos (1A) e parábolas (1B) como curvas de nível. Agora vamos completar o trio de cônicas clássicas: uma elipse. Você já encontrou elipses quando estudou cônicas — aqui, cada valor de \(c\) define uma elipse diferente, como seções horizontais de uma cratera.

    O perfil de uma cratera elíptica é modelado por \(f(x, y) = \frac{x^2}{4} + y^2\). Determine a forma das curvas de nível \(f(x, y) = c\) para \(c > 0\) e identifique os semi-eixos para o caso \(c = 1\).

    ARQUIVO AUXILIAR [SIGMA-N]
    [NOTA TÉCNICA]: A equação \(\frac{x^2}{4} + y^2 = c\) pode ser reescrita como \(\frac{x^2}{(2\sqrt{c})^2} + \frac{y^2}{(\sqrt{c})^2} = 1\), que é a forma padrão de uma elipse com centro na origem.
    RELATÓRIO DE CÁLCULO [SIGMA-N REQUERIDO]

    Solução completa:

    Para as curvas de nível, fixamos \(f(x, y) = c\): \[ \frac{x^2}{4} + y^2 = c \] Dividindo ambos os lados por \(c\) (com \(c > 0\)): \[ \frac{x^2}{4c} + \frac{y^2}{c} = 1 \] Reescrevendo na forma padrão da elipse: \[ \frac{x^2}{(2\sqrt{c})^2} + \frac{y^2}{(\sqrt{c})^2} = 1 \] As curvas de nível são elipses centradas na origem com:

    • Semi-eixo maior (horizontal): \(a = 2\sqrt{c}\)
    • Semi-eixo menor (vertical): \(b = \sqrt{c}\)
    Para \(c = 1\): \[ \frac{x^2}{4} + y^2 = 1 \] Semi-eixo maior: \(a = 2\) (direção \(x\))
    Semi-eixo menor: \(b = 1\) (direção \(y\))

    A razão \(a/b = 2\) indica que a cratera é duas vezes mais alongada na direção \(x\) do que na direção \(y\).
