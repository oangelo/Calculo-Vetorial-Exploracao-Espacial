• EXERCÍCIO VECTOR-1A [NÍVEL ALFA]
    ORIGEM: Programa de Mapeamento Lunar - Projeto Ranger, 1962

    Uma sonda de reconhecimento transmite a seguinte equação para a superfície de uma cratera: \(f(x, y) = x^2 + y^2\). Identifique o formato das curvas de nível \(f(x, y) = c\) para \(c > 0\). Que tipo de curva geométrica representa cada nível?

    ARQUIVO AUXILIAR [SIGMA-N]
    [NOTA TÉCNICA]: A equação \(x^2 + y^2 = c\) com \(c > 0\) é reconhecida como a equação padrão de uma figura geométrica fundamental no plano cartesiano. Calcule o raio como \(r = \sqrt{c}\).
    RELATÓRIO DE CÁLCULO [SIGMA-N REQUERIDO]

    Solução completa:

    Para encontrar as curvas de nível, fixamos \(f(x,y) = c\): \[ x^2 + y^2 = c \] Quando \(c > 0\), podemos reescrever como: \[ x^2 + y^2 = (\sqrt{c})^2 \] Esta é a equação de um círculo centrado na origem \((0, 0)\) com raio \(r = \sqrt{c}\).

    Para \(c = 1\): círculo de raio 1
    Para \(c = 4\): círculo de raio 2
    Para \(c = 9\): círculo de raio 3

    As curvas de nível são círculos concêntricos centrados na origem. A distância entre círculos consecutivos (com \(c\) igualmente espaçado) diminui conforme \(c\) aumenta, indicando que a inclinação da superfície aumenta com a distância da origem.
