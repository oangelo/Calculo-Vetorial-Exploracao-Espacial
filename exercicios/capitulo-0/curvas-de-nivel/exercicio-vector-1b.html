• EXERCÍCIO VECTOR-1B [NÍVEL ALFA]
    ORIGEM: Análise de Terreno - Corredor de Berlim, 1961

    Um modelo de elevação de terreno é dado por \(f(x, y) = y - x^2\). Determine o formato das curvas de nível \(f(x, y) = c\). Para qual valor de \(c\) a curva de nível passa pela origem?

    ARQUIVO AUXILIAR [SIGMA-N]
    [NOTA TÉCNICA]: Isolando \(y\), obtemos \(y = x^2 + c\). Esta é a equação de uma parábola com vértice em \((0, c)\) que abre para cima. Substitua \((0, 0)\) para encontrar o valor de \(c\).
    RELATÓRIO DE CÁLCULO [SIGMA-N REQUERIDO]

    Solução completa:

    Para as curvas de nível, fixamos \(f(x, y) = c\): \[ y - x^2 = c \] Isolando \(y\): \[ y = x^2 + c \] Esta é a equação de uma parábola com vértice no ponto \((0, c)\) e eixo de simetria ao longo do eixo \(y\).

    Para \(c = -1\): parábola com vértice em \((0, -1)\)
    Para \(c = 0\): parábola com vértice na origem \(y = x^2\)
    Para \(c = 1\): parábola com vértice em \((0, 1)\)

    Passando pela origem: Substituindo \((x,y) = (0,0)\): \[ 0 - 0^2 = c \implies c = 0 \] A curva de nível que passa pela origem é \(y = x^2\), a parábola com vértice na própria origem.
