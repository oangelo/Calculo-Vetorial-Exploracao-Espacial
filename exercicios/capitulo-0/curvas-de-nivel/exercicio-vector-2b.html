• EXERCÍCIO VECTOR-2B [NÍVEL BETA]
    ORIGEM: Estudo de Pressão Atmosférica - Balões Estratosféricos, 1963

    A variação de pressão em uma região é modelada por \(f(x, y) = e^{-(x^2 + y^2)}\). Determine a equação da curva de nível que passa pelo ponto \((1, 0)\). O que acontece com as curvas de nível quando \(c\) se aproxima de 1?

    ARQUIVO AUXILIAR [SIGMA-N]
    [NOTA TÉCNICA]: Calcule \(c = f(1,0) = e^{-1}\). Para resolver \(e^{-(x^2+y^2)} = c\), aplique o logaritmo natural em ambos os lados. Note que \(c \to 1\) quando \(x^2 + y^2 \to 0\).
    RELATÓRIO DE CÁLCULO [SIGMA-N REQUERIDO]

    Solução completa:

    Passo 1: Calcular \(c\): \[ c = f(1, 0) = e^{-(1^2 + 0^2)} = e^{-1} = \frac{1}{e} \approx 0{,}368 \] Passo 2: Encontrar a curva de nível genérica: \[ e^{-(x^2 + y^2)} = c \] Aplicando \(\ln\) em ambos os lados (com \(0 < c \leq 1\)): \[ -(x^2 + y^2) = \ln c \] \[ x^2 + y^2 = -\ln c \] Passo 3: Para \(c = e^{-1}\): \[ x^2 + y^2 = -\ln(e^{-1}) = 1 \] A curva de nível é um círculo de raio 1 centrado na origem.

    Quando \(c \to 1\):
    Como \(-\ln c \to 0\) quando \(c \to 1\), o raio do círculo \(r = \sqrt{-\ln c} \to 0\). As curvas de nível se contraem em torno da origem, indicando que o ponto \((0,0)\) é um máximo global da função com valor \(f(0,0) = 1\).
