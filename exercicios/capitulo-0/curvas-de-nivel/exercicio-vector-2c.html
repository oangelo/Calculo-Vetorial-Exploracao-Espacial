• EXERCÍCIO VECTOR-2C [NÍVEL BETA]
    ORIGEM: Simulação de Trajetória - Projeto Gemini, 1965

    Uma função descreve a altitude de uma trajetória de reentrada: \(f(x, y) = 3x + 4y\). Encontre a equação da curva de nível que passa pelo ponto \((2, -1)\). Que tipo de curva é esta e qual é sua inclinação?

    ARQUIVO AUXILIAR [SIGMA-N]
    [NOTA TÉCNICA]: Calcule \(c = f(2, -1) = 6 - 4 = 2\). A equação \(3x + 4y = 2\) é uma reta. Reescreva na forma \(y = mx + b\) para identificar a inclinação \(m\).
    RELATÓRIO DE CÁLCULO [SIGMA-N REQUERIDO]

    Solução completa:

    Passo 1: Calcular \(c\): \[ c = f(2, -1) = 3(2) + 4(-1) = 6 - 4 = 2 \] Passo 2: Equação da curva de nível: \[ 3x + 4y = 2 \] Passo 3: Identificar o tipo e a inclinação. Isolando \(y\): \[ 4y = 2 - 3x \] \[ y = -\frac{3}{4}x + \frac{1}{2} \] A curva de nível é uma reta com:

    • Inclinação: \(m = -\frac{3}{4}\)
    • Intercepto-\(y\): \(b = \frac{1}{2}\)
    • Intercepto-\(x\): \(x = \frac{2}{3}\)
    Observação: Como \(f(x,y) = 3x + 4y\) é uma função linear, TODAS as suas curvas de nível são retas paralelas com inclinação \(-\frac{3}{4}\). O gradiente é constante: \(\nabla f = (3, 4)\), perpendicular a todas as curvas de nível.
