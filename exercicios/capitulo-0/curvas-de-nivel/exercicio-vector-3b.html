• EXERCÍCIO VECTOR-3B [NÍVEL GAMMA]
    ORIGEM: Análise de Relevo - Zona de Pouso de Emergência, 1964

    Considere a função \(f(x, y) = \sin(x) \cdot \sin(y)\). Esboce as curvas de nível para \(c = 0\), \(c = 0{,}5\) e \(c = -0{,}5\) na região \(-\pi \leq x \leq \pi\), \(-\pi \leq y \leq \pi\). Quantos pontos de máximo local existem nesta região?

    ARQUIVO AUXILIAR [SIGMA-N]
    [NOTA TÉCNICA]: Para \(c = 0\): \(\sin(x)\sin(y) = 0\), ou seja, \(\sin(x) = 0\) OU \(\sin(y) = 0\). Os máximos ocorrem quando ambos \(\sin(x) = 1\) e \(\sin(y) = 1\), isto é, em \((\pi/2, \pi/2)\) e combinações de sinais.
    RELATÓRIO DE CÁLCULO [SIGMA-N REQUERIDO]

    Solução completa:

    Curva de nível \(c = 0\): \[ \sin(x)\sin(y) = 0 \implies \sin(x) = 0 \text{ ou } \sin(y) = 0 \] Isto dá as retas \(x = -\pi, 0, \pi\) e \(y = -\pi, 0, \pi\). A curva \(c = 0\) forma uma grade de linhas horizontais e verticais.

    Curva de nível \(c = 0{,}5\): \[ \sin(x)\sin(y) = \frac{1}{2} \] Como \(|\sin| \leq 1\), precisamos de ambos \(\sin(x)\) e \(\sin(y)\) com mesmo sinal. As curvas são fechadas ao redor dos pontos de máximo em \((\pi/2, \pi/2)\) e \((-\pi/2, -\pi/2)\).

    Curva de nível \(c = -0{,}5\): \[ \sin(x)\sin(y) = -\frac{1}{2} \] Curvas fechadas ao redor dos pontos de mínimo em \((\pi/2, -\pi/2)\) e \((-\pi/2, \pi/2)\).

    Pontos de máximo local:
    Os máximos \(f = 1\) ocorrem quando \(\sin(x) = \pm 1\) e \(\sin(y) = \pm 1\) com mesmo sinal:

    • \((\pi/2, \pi/2)\): \(f = (+1)(+1) = 1\)
    • \((-\pi/2, -\pi/2)\): \(f = (-1)(-1) = 1\)
    Existem 2 pontos de máximo local com valor 1. Similarmente, 2 pontos de mínimo local com valor \(-1\).
