• EXERCÍCIO VECTOR-3C [NÍVEL GAMMA]
    ORIGEM: Mapeamento de Densidade Populacional - Guerra Fria Civil, 1963

    A função \(f(x, y) = \frac{1}{1 + x^2 + y^2}\) modela a densidade de um sinal centrado na origem. Esboce as curvas de nível para \(c = \frac{1}{2}\), \(c = \frac{1}{5}\) e \(c = \frac{1}{10}\). O que representa o valor \(c = 1\)?

    ARQUIVO AUXILIAR [SIGMA-N]
    [NOTA TÉCNICA]: Resolva \(\frac{1}{1+x^2+y^2} = c\) isolando \(x^2 + y^2\). Obterá \(x^2 + y^2 = \frac{1}{c} - 1\). Para cada \(c\), calcule o raio correspondente.
    RELATÓRIO DE CÁLCULO [SIGMA-N REQUERIDO]

    Solução completa:

    Passo 1: Encontrar a equação geral: \[ \frac{1}{1 + x^2 + y^2} = c \] \[ 1 + x^2 + y^2 = \frac{1}{c} \] \[ x^2 + y^2 = \frac{1}{c} - 1 \] As curvas de nível são círculos centrados na origem com raio \(r = \sqrt{\frac{1}{c} - 1}\).

    Passo 2: Calcular raios específicos:

    • \(c = \frac{1}{2}\): \(r = \sqrt{2 - 1} = 1\)
    • \(c = \frac{1}{5}\): \(r = \sqrt{5 - 1} = 2\)
    • \(c = \frac{1}{10}\): \(r = \sqrt{10 - 1} = 3\)
    Passo 3: Interpretação de \(c = 1\): \[ x^2 + y^2 = \frac{1}{1} - 1 = 0 \implies x = y = 0 \] A curva de nível \(c = 1\) é apenas o ponto origem \((0, 0)\), que é o máximo global da função. O sinal é mais intenso no centro e decai com a distância radial.
