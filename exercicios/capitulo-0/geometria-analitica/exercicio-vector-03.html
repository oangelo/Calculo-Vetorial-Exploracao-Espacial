• EXERCÍCIO VECTOR-03 ALFA
    ORIGEM: Combinação de Deslocamentos — Simulação de Missão [SIGMA-1]
    Nos exercícios anteriores, você trabalhou com um único vetor. Agora vamos operar com dois: somar, combinar e normalizar. Estas operações são a base para tudo que vem depois — inclusive o produto escalar no próximo bloco.

    Dois deslocamentos consecutivos se combinam por soma. Um deslocamento repetido se multiplica por escalar.

    Dados \(\vec{u} = (1, 2, 0)\) e \(\vec{v} = (3, -1, 4)\), calcule:

    (a) \(\vec{u} + \vec{v}\).
    (b) \(3\vec{u} - 2\vec{v}\).
    (c) O vetor unitário na direção de \(\vec{u}\).

    ARQUIVO AUXILIAR [SIGMA-1]
    [NOTA TÉCNICA]: Operações componente a componente. Para o vetor unitário, divida o vetor pela sua norma (que você calculou no exercício anterior).
    RELATÓRIO DE CÁLCULO [SIGMA-1 REQUERIDO]

    PROTOCOLO DE OPERAÇÕES VETORIAIS:

    (a) \(\vec{u} + \vec{v} = (1+3,\;2+(-1),\;0+4) = (4, 1, 4)\).

    (b) \(3\vec{u} - 2\vec{v} = (3,6,0) - (6,-2,8) = (-3, 8, -8)\).

    (c) \(\\|\vec{u}\\| = \sqrt{1+4+0} = \sqrt{5}\).
    Vetor unitário: \(\hat{u} = \dfrac{\vec{u}}{\\|\vec{u}\\|} = \left(\dfrac{1}{\sqrt{5}},\;\dfrac{2}{\sqrt{5}},\;0\right)\).
