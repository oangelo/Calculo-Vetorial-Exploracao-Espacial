• EXERCÍCIO VECTOR-04 GAMMA
    ORIGEM: Análise de Velocidade — Direções de Interceptação [SIGMA-2]
    Agora que sabemos operar vetores individualmente, vamos ver como DOIS vetores interagem. O produto escalar mede o quanto dois vetores apontam na mesma direção — e dele derivamos o ângulo entre eles. Dois satélites com velocidades \\(\\vec{u}\\) e \\(\\vec{v}\\): estão se aproximando ou se afastando? O produto escalar responde.

    Dados \\(\\vec{u} = (2, -1, 3)\\) e \\(\\vec{v} = (1, 4, -2)\\):

    (a) Calcule o produto escalar \\(\\vec{u} \\cdot \\vec{v}\\).
    (b) Calcule o ângulo entre eles (use as normas dos vetores).
    (c) Os vetores são perpendiculares? Justifique.

    ARQUIVO AUXILIAR [SIGMA-2]
    [NOTA TÉCNICA]: O produto escalar é a soma dos produtos das componentes correspondentes. Para o ângulo, use \\(\\cos\\theta = \\dfrac{\\vec{u}\\cdot\\vec{v}}{\\|\\vec{u}\\|\\,\\|\\vec{v}\\|}\\) — lembre-se do cálculo de norma do VECTOR-02. Se o produto escalar é zero, os vetores são perpendiculares (ângulo de 90°).
    RELATÓRIO DE CÁLCULO [SIGMA-2 REQUERIDO]

    PROTOCOLO DE PRODUTO ESCALAR:

    (a) \\(\\vec{u} \\cdot \\vec{v} = 2(1) + (-1)(4) + 3(-2) = 2 - 4 - 6 = -8\\).

    (b) \\(\\|\\vec{u}\\| = \\sqrt{4+1+9} = \\sqrt{14}\\), \\(\\|\\vec{v}\\| = \\sqrt{1+16+4} = \\sqrt{21}\\).
    \\(\\cos\\theta = \\dfrac{-8}{\\sqrt{14}\\cdot\\sqrt{21}} = \\dfrac{-8}{\\sqrt{294}} = \\dfrac{-8}{7\\sqrt{6}}\\).
    \\(\\theta = \\arccos\\left(\\dfrac{-8}{7\\sqrt{6}}\\right) \\approx 118°\\).

    (c) Não são perpendiculares, pois \\(\\vec{u}\\cdot\\vec{v} = -8 \\neq 0\\).
