• EXERCÍCIO VECTOR-05 BETA
    ORIGEM: Decomposição de Forças — Análise de Trajetória [SIGMA-2]
    No exercício anterior, usamos o produto escalar para calcular o ângulo entre vetores. Agora vamos usá-lo para algo mais prático: decompor um vetor em componente na direção de outro e componente perpendicular. Isso é a projeção ortogonal — a "sombra" de um vetor sobre a direção de outro.

    Dado \\(\\vec{v} = (3, 4, 0)\\) e \\(\\vec{u} = (1, 0, 0)\\):

    (a) Encontre a projeção de \\(\\vec{v}\\) sobre \\(\\vec{u}\\).
    (b) Encontre o componente perpendicular.
    (c) Verifique que os dois componentes são ortogonais (use o produto escalar).

    ARQUIVO AUXILIAR [SIGMA-2]
    [NOTA TÉCNICA]: \\(\\text{proj}_{\\vec{u}}\\vec{v} = \\dfrac{\\vec{v}\\cdot\\vec{u}}{\\|\\vec{u}\\|^2}\\,\\vec{u}\\). O perpendicular é \\(\\vec{v} - \\text{proj}_{\\vec{u}}\\vec{v}\\). Para verificar ortogonalidade, basta mostrar que o produto escalar é zero (como no VECTOR-04).
    RELATÓRIO DE CÁLCULO [SIGMA-2 REQUERIDO]

    PROTOCOLO DE PROJEÇÃO:

    (a) \\(\\vec{v}\\cdot\\vec{u} = 3\\), \\(\\|\\vec{u}\\|^2 = 1\\).
    \\(\\text{proj}_{\\vec{u}}\\vec{v} = \\dfrac{3}{1}(1,0,0) = (3, 0, 0)\\).

    (b) \\(\\vec{v}_{\\perp} = (3,4,0) - (3,0,0) = (0, 4, 0)\\).

    (c) \\((3,0,0)\\cdot(0,4,0) = 0\\). Confirmado: são ortogonais.
