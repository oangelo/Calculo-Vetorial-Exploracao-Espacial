• EXERCÍCIO VECTOR-07
    ORIGEM: Cálculo de Volume - Projeto de Estação Orbital [SIGMA-3]

    Três vetores definem um paralelepípedo. O produto misto \(\vec{a}\cdot(\vec{b}\times\vec{c})\) dá o volume (com sinal).

    Dados \(\vec{a} = (1, 2, 3)\), \(\vec{b} = (0, 1, 1)\) e \(\vec{c} = (2, 0, 1)\):

    (a) Calcule o produto misto \(\vec{a}\cdot(\vec{b}\times\vec{c})\).
    (b) Calcule o volume do paralelepípedo.
    (c) Repita para \(\vec{a}=(1,0,0)\), \(\vec{b}=(0,1,0)\), \(\vec{c}=(0,0,1)\). O resultado é esperado? Por quê?

    ARQUIVO AUXILIAR [SIGMA-3]
    [NOTA TÉCNICA]: O produto misto é o determinante da matriz \(3\times 3\) formada pelas componentes. O valor absoluto é o volume.
    RELATÓRIO DE CÁLCULO [SIGMA-3 REQUERIDO]

    PROTOCOLO DE PRODUTO MISTO:

    (a) \(\vec{b}\times\vec{c} = (1, 2, -2)\).
    \(\vec{a}\cdot(\vec{b}\times\vec{c}) = 1(1) + 2(2) + 3(-2) = 1 + 4 - 6 = -1\).

    (b) Volume = \(|-1| = 1\).

    (c) \(\vec{b}\times\vec{c} = (1,0,0)\), produto misto = \(1\).
    Sim, o volume do cubo unitário é 1, como esperado.
