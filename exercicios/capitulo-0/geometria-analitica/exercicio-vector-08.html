• EXERCÍCIO VECTOR-08
    ORIGEM: Trajetória de Interceptação - Centro de Comando [SIGMA-3]

    Dois pontos definem uma direção (vetor diretor) e um ponto inicial. Isso é tudo que precisamos para descrever uma reta.

    Dados \(P(1, 0, 2)\) e \(Q(2, 3, 4)\):

    (a) Encontre o vetor diretor \(\vec{PQ}\).
    (b) Escreva a equação vetorial da reta.
    (c) Escreva as equações paramétricas \(x(t), y(t), z(t)\).

    ARQUIVO AUXILIAR [SIGMA-3]
    [NOTA TÉCNICA]: O vetor diretor é \(\vec{PQ}\). A equação vetorial é \(\vec{r}(t) = \vec{P} + t\,\vec{PQ}\).
    RELATÓRIO DE CÁLCULO [SIGMA-3 REQUERIDO]

    PROTOCOLO DE EQUAÇÃO DE RETA:

    (a) \(\vec{d} = \vec{PQ} = (2-1,\;3-0,\;4-2) = (1, 3, 2)\).

    (b) \(\vec{r}(t) = (1,0,2) + t(1,3,2)\).

    (c) \(x(t) = 1 + t\), \(y(t) = 3t\), \(z(t) = 2 + 2t\).
