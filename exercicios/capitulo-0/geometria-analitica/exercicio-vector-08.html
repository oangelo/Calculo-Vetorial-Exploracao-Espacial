• EXERCÍCIO VECTOR-08 GAMMA
    ORIGEM: Trajetória de Interceptação — Centro de Comando [SIGMA-3]
    Agora vamos usar vetores para descrever trajetórias. Dois pontos no espaço definem uma reta: o primeiro ponto dá a posição inicial, o vetor entre eles dá a direção. Tudo que fizemos até aqui — vetores, norma, soma — converge para este momento.

    Dados \(P(1, 0, 2)\) e \(Q(2, 3, 4)\):

    (a) Encontre o vetor diretor \(\vec{PQ}\).
    (b) Escreva a equação vetorial da reta: \(\vec{r}(t) = \vec{P} + t\,\vec{PQ}\).
    (c) Escreva as equações paramétricas \(x(t), y(t), z(t)\).

    ARQUIVO AUXILIAR [SIGMA-3]
    [NOTA TÉCNICA]: O vetor diretor é \(\vec{PQ}\) — o mesmo tipo de cálculo do VECTOR-01. A equação vetorial é \(\vec{r}(t) = \vec{P} + t\,\vec{PQ}\). Para a paramétrica, extraia cada componente separadamente.
    RELATÓRIO DE CÁLCULO [SIGMA-3 REQUERIDO]

    PROTOCOLO DE EQUAÇÃO DE RETA:

    (a) \(\vec{d} = \vec{PQ} = (2-1,\;3-0,\;4-2) = (1, 3, 2)\).

    (b) \(\vec{r}(t) = (1,0,2) + t(1,3,2)\).

    (c) \(x(t) = 1 + t\), \(y(t) = 3t\), \(z(t) = 2 + 2t\).
