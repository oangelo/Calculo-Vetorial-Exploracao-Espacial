• EXERCÍCIO VECTOR-09 ALFA
    ORIGEM: Navegação Orbital — Formas Alternativas de Trajetória [SIGMA-3]
    No exercício anterior, escrevemos a reta na forma vetorial. Mas a mesma reta admite várias representações: paramétrica, simétrica. Cada uma tem vantagens — a simétrica é compacta, a paramétrica facilita substituir em planos. Reconhecer formas equivalentes é uma habilidade essencial.

    Dada a reta \\(\\vec{r}(t) = (1, 1, 0) + t(2, 1, 1)\\):

    (a) Escreva as equações paramétricas.
    (b) Escreva as equações simétricas (eliminando o parâmetro \\(t\\)).
    (c) Encontre dois pontos da reta (para \\(t=0\\) e \\(t=1\\)).

    ARQUIVO AUXILIAR [SIGMA-3]
    [NOTA TÉCNICA]: Para a paramétrica, extraia \\(x(t), y(t), z(t)\\). Para a simétrica, isola \\(t\\) em cada equação e iguale. Se alguma componente do diretor for zero (não é o caso aqui), use a forma por pares de equações.
    RELATÓRIO DE CÁLCULO [SIGMA-3 REQUERIDO]

    PROTOCOLO DE FORMAS EQUIVALENTES:

    (a) \\(x = 1 + 2t\\), \\(y = 1 + t\\), \\(z = t\\).

    (b) De \\(x=1+2t\\): \\(t = \\dfrac{x-1}{2}\\).
    De \\(y=1+t\\): \\(t = y-1\\).
    De \\(z=t\\): \\(t = z\\).
    Simétrica: \\(\\dfrac{x-1}{2} = y-1 = z\\).

    (c) Para \\(t=0\\): ponto \\((1,1,0)\\). Para \\(t=1\\): ponto \\((3,2,1)\\).
