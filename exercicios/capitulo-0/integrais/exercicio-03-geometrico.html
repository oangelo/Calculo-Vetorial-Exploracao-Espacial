• EXERCÍCIO 3 [NÍVEL ALFA]
    ORIGEM: Seção de Análise Geométrica - Projeto de Fundamentos [MAI/1962]

    Os dois exercícios anteriores envolviam funções lineares. Agora vamos para uma função quadrática, que gera uma região sob uma parábola. Este exercício motiva: se não conhecêssemos a antiderivada, como calcularíamos essa área?

    Calcule a integral definida usando o TFC e confira o significado geométrico:

    \[\int_0^{1} x^2\,dx\]

    (a) Use o Teorema Fundamental do Cálculo para encontrar o valor da integral. (Dica: qual é a antiderivada de \(x^2\)? Lembre-se da derivação.)

    (b) Esboce o gráfico de \(f(x) = x^2\) no intervalo \([0, 1]\). A região sob a curva é côncava para cima — ela é maior ou menor que o triângulo de vértices \((0,0)\), \((1,0)\), \((1,1)\)?

    (c) Esse exercício mostra por que precisamos de técnicas de integração: sem conhecer a antiderivada, calcular essa área exigiria somas de Riemann (limite de somas). As técnicas que veremos a seguir nos dão atalhos para encontrar antiderivadas.

    RELATÓRIO DE CÁLCULO [SIGMA-4 REQUERIDO]

    (a) Uma antiderivada de \(x^2\) é \(F(x) = \frac{x^3}{3}\). Pelo TFC: \(\int_0^1 x^2\,dx = \frac{1}{3} - 0 = \frac{1}{3}.\)

    (b) A curva \(y = x^2\) passa pela origem e por \((1, 1)\). O triângulo de vértices \((0,0)\), \((1,0)\), \((1,1)\) tem área \(\frac{1}{2} \times 1 \times 1 = \frac{1}{2}\). Como a parábola é côncava para cima, a área sob ela (\(\frac{1}{3}\)) é menor que a área do triângulo (\(\frac{1}{2}\)). Isso faz sentido: a parábola fica abaixo da reta \(y = x\) no intervalo \((0,1)\).

    (c) Este exercício mostra que o TFC nos dá um atalho poderoso: em vez de calcular o limite de uma soma infinita (somas de Riemann), basta encontrar uma antiderivada. As técnicas a seguir nos ensinam como encontrar antiderivadas quando não são óbvias.
