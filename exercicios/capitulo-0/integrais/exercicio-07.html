• EXERCÍCIO 7 [NÍVEL GAMMA]
    ORIGEM: Setor de Substituições Trigonométricas [AGO/1962]

    Esta integral envolve uma raiz quadrada de uma expressão quadrática. O formato \(\sqrt{a^2 - x^2}\) sugere uma substituição trigonométrica — a mesma técnica que resolveria o problema do semicírculo (Ex. 4 da Seção I). Vamos guiá-la passo a passo.

    Calcule a integral, onde \(a\) é constante positiva:

    \[\int \sqrt{a^2 - x^2}\,dx\]

    (a) Faça a substituição \(x = a\sin(\theta)\). Encontre \(dx\) em termos de \(\theta\).

    (b) Simplifique \(\sqrt{a^2 - x^2}\) em termos de \(\theta\). (Dica: use \(\sin^2(\theta) + \cos^2(\theta) = 1\).)

    (c) A integral em \(\theta\) é \(\int a^2\cos^2(\theta)\,d\theta\). Resolva-a usando a identidade de potência.

    (d) Converta o resultado de volta para \(x\), usando \(\theta = \arcsin(x/a)\).

    ARQUIVO AUXILIAR [SIGMA-1]
    [NOTA TÉCNICA]: Use \(x = a\sin(\theta)\). Então \(\sqrt{a^2 - x^2} = a\cos(\theta)\).
    RELATÓRIO DE CÁLCULO [SIGMA-4 REQUERIDO]

    (a) Com \(x = a\sin(\theta)\), temos \(dx = a\cos(\theta)\,d\theta\).

    (b) \(\sqrt{a^2 - x^2} = \sqrt{a^2 - a^2\sin^2(\theta)} = \sqrt{a^2\cos^2(\theta)} = a\cos(\theta)\) (para \(\theta \in [-\pi/2, \pi/2]\)).

    (c) A integral vira: \(\int a\cos(\theta)\cdot a\cos(\theta)\,d\theta = a^2\int\cos^2(\theta)\,d\theta.\) Usando \(\cos^2(\theta) = \frac{1 + \cos(2\theta)}{2}\): \(\frac{a^2}{2}\int(1 + \cos(2\theta))\,d\theta = \frac{a^2}{2}\left(\theta + \frac{\sin(2\theta)}{2}\right) + C.\)

    (d) Como \(\theta = \arcsin(x/a)\), temos \(\sin(\theta) = x/a\) e \(\cos(\theta) = \sqrt{1 - x^2/a^2} = \frac{\sqrt{a^2 - x^2}}{a}\). Além disso \(\sin(2\theta) = 2\sin(\theta)\cos(\theta) = \frac{2x\sqrt{a^2 - x^2}}{a^2}\).

    Resultado final: \(\frac{a^2}{2}\arcsin\!\left(\frac{x}{a}\right) + \frac{x}{2}\sqrt{a^2 - x^2} + C.\)
