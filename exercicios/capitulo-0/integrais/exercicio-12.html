• EXERCÍCIO 12 [NÍVEL GAMMA]
    ORIGEM: Substituições Avançadas - Seção Hiperbólica [SET/1962]

    Esta integral envolve \(\sqrt{x^2 + a^2}\), o terceiro formato clássico de substituição trigonométrica. A técnica gera uma integral de \(\sec^3(\theta)\), que é resolvida por partes. Vamos guiá-la.

    Calcule a integral, onde \(a\) é constante positiva:

    \[\int \sqrt{x^2 + a^2}\,dx\]

    (a) Faça a substituição \(x = a\tan(\theta)\). Encontre \(dx\) e simplifique \(\sqrt{x^2 + a^2}\) em termos de \(\theta\).

    (b) A integral resultante é \(\int a^2\sec^3(\theta)\,d\theta\). Escreva \(\sec^3(\theta) = \sec(\theta)\cdot\sec^2(\theta)\) e aplique integração por partes.

    (c) A integração por partes gera uma equação com \(I = \int\sec^3(\theta)\,d\theta\) dos dois lados. Resolva essa equação para \(I\).

    (d) Converta o resultado de volta para \(x\).

    ARQUIVO AUXILIAR [SIGMA-1]
    [NOTA TÉCNICA]: Pode-se usar \(x = a\tan(\theta)\) ou \(x = a\sinh(t)\). Observe o retorno para a variável original.
    RELATÓRIO DE CÁLCULO [SIGMA-4 REQUERIDO]

    (a) Com \(x = a\tan(\theta)\): \(\sqrt{x^2 + a^2} = a\sec(\theta)\) e \(dx = a\sec^2(\theta)\,d\theta\).

    (b) A integral fica \(a^2\int\sec^3(\theta)\,d\theta\). Por partes com \(u = \sec(\theta)\), \(dv = \sec^2(\theta)\,d\theta\): \(du = \sec(\theta)\tan(\theta)\,d\theta\), \(v = \tan(\theta)\).

    \(I = \sec(\theta)\tan(\theta) - \int\sec(\theta)\tan^2(\theta)\,d\theta\)

    (c) Usando \(\tan^2(\theta) = \sec^2(\theta) - 1\): \(I = \sec(\theta)\tan(\theta) - I + \int\sec(\theta)\,d\theta\). Portanto \(2I = \sec(\theta)\tan(\theta) + \ln|\sec(\theta) + \tan(\theta)|\).

    \(\int\sec^3(\theta)\,d\theta = \frac{1}{2}\sec(\theta)\tan(\theta) + \frac{1}{2}\ln|\sec(\theta) + \tan(\theta)| + C.\)

    (d) Substituindo de volta para \(x\): \(\frac{x}{2}\sqrt{x^2 + a^2} + \frac{a^2}{2}\ln\bigl|x + \sqrt{x^2 + a^2}\bigr| + C.\)
