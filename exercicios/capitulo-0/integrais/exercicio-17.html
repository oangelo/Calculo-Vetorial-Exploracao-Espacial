• EXERCÍCIO 17 [NÍVEL GAMMA]
    ORIGEM: Arquivos de Frações Parciais Quadráticas [SET/1962]

    Esta integral combina frações parciais com completar quadrado. A estratégia é: decompor o numerador, separar em duas integrais, e tratar a segunda com completar quadrado e arctangente.

    Calcule a integral:

    \[\int \frac{2x + 3}{x^2 + x + 1}\,dx\]

    (a) Observe que a derivada do denominador é \(2x + 1\). Reescreva o numerador como \((2x + 1) + 2\) e separe em duas integrais.

    (b) A primeira integral gera um logaritmo. Para a segunda, complete o quadrado no denominador: \(x^2 + x + 1 = (x + \frac{1}{2})^2 + \frac{3}{4}\).

    (c) A segunda integral tem a forma \(\int\frac{du}{u^2 + a^2}\). Resolva usando arctangente.

    ARQUIVO AUXILIAR [SIGMA-1]
    [NOTA TÉCNICA]: Observe que a derivada de \(x^2 + x + 1\) é \(2x + 1\). Separe o numerador para usar essa informação.
    RELATÓRIO DE CÁLCULO [SIGMA-4 REQUERIDO]

    (a) \(\frac{2x+3}{x^2 + x + 1} = \frac{(2x+1) + 2}{x^2 + x + 1} = \frac{2x+1}{x^2 + x + 1} + \frac{2}{x^2 + x + 1}.\)

    (b) A primeira parte gera \(\ln|x^2 + x + 1|\). Para a segunda, complete o quadrado: \(x^2 + x + 1 = (x+\tfrac{1}{2})^2 + \tfrac{3}{4}\). Seja \(u = x + \tfrac{1}{2}\), \(a^2 = \tfrac{3}{4}\).

    (c) \(\int\frac{2}{u^2 + a^2}\,du = \frac{2}{a} \arctan\!\left(\frac{u}{a}\right) = \frac{2}{\sqrt{3}/2} \arctan\!\left(\frac{x+1/2}{\sqrt{3}/2}\right) = \frac{4}{\sqrt{3}} \arctan\!\left(\frac{2x+1}{\sqrt{3}}\right).\)

    Resultado final: \(\ln|x^2 + x + 1| + \frac{4}{\sqrt{3}}\arctan\!\left(\frac{2x+1}{\sqrt{3}}\right) + C.\)
