    ORIGEM: Substituições Trigonométricas - Equipe de Aplicações [OUT/1962]

    Antes de irmos para os exercícios OMEGA, vamos praticar uma substituição trigonométrica mais direta. A substituição de Weierstrass (\(t = \tan(x/2)\)) é uma técnica avançada que fica para estudos futuros. Em vez disso, vamos resolver uma integral trigonométrica com substituição simples.

    Calcule a integral:

    \[\int \frac{\sin(x)}{1 + \cos(x)}\,dx\]

    ARQUIVO AUXILIAR [SIGMA-1]
    [NOTA TÉCNICA]: Observe que a derivada de \(1 + \cos(x)\) é \(-\sin(x)\). Tente \(u = 1 + \cos(x)\).
    RELATÓRIO DE CÁLCULO [SIGMA-4 REQUERIDO]

    Seja \(u = 1 + \cos(x)\). Então \(du = -\sin(x)\,dx\), ou seja \(\sin(x)\,dx = -du\).

    \(\int \frac{\sin(x)}{1 + \cos(x)}\,dx = \int \frac{-du}{u} = -\ln|u| + C = -\ln|1 + \cos(x)| + C.\)
