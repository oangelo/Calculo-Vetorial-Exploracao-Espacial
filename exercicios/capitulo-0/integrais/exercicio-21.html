• EXERCÍCIO 21 [NÍVEL OMEGA]
    ORIGEM: Análise Orbital Completa - Centro de Cálculos [CLASSIFICADO]

    Calcule a integral:

    \[\int \frac{x^3}{\sqrt{x^2 + 4}}\,dx\]

    RELATÓRIO DE CÁLCULO [SIGMA-4 REQUERIDO]

    Substitua \(u = x^2 + 4\), então \(du = 2x\,dx\) e \(x^2 = u - 4\).

    \(\displaystyle\int \frac{x^3}{\sqrt{x^2+4}}\,dx = \int \frac{x^2 \cdot x}{\sqrt{x^2+4}}\,dx = \frac{1}{2}\int \frac{u-4}{\sqrt{u}}\,du = \frac{1}{2}\int \left(u^{1/2} - 4u^{-1/2}\right)du\).

    \[ = \frac{1}{2}\left(\frac{2}{3}u^{3/2} - 8u^{1/2}\right) + C = \frac{1}{3}u^{3/2} - 4u^{1/2} + C \]

    Substituindo \(u = x^2+4\): \[ = \frac{1}{3}(x^2+4)^{3/2} - 4\sqrt{x^2+4} + C = \frac{\sqrt{x^2+4}}{3}(x^2 - 8) + C. \]
