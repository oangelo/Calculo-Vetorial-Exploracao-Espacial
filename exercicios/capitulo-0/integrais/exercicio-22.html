• EXERCÍCIO 22 [NÍVEL OMEGA]
    ORIGEM: Modelagem de Resistência - Projeto de Reentrada [SIGMA-4]

    Calcule a integral:

    \[\int \frac{x^2 + 1}{x^2 - x - 2}\,dx\]

    RELATÓRIO DE CÁLCULO [SIGMA-4 REQUERIDO]

    Como o grau do numerador é igual ao do denominador, primeiro faça a divisão polinomial (ou some e subtraia):

    \(\dfrac{x^2+1}{x^2-x-2} = 1 + \dfrac{x+3}{x^2-x-2}\).

    Fatorando: \(x^2-x-2 = (x-2)(x+1)\).

    Frações parciais: \(\dfrac{x+3}{(x-2)(x+1)} = \dfrac{A}{x-2} + \dfrac{B}{x+1}\).

    \(x+3 = A(x+1) + B(x-2)\). Para \(x=2\): \(5=3A \Rightarrow A=\frac{5}{3}\). Para \(x=-1\): \(2=-3B \Rightarrow B=-\frac{2}{3}\).

    \[ \int \left(1 + \frac{5/3}{x-2} - \frac{2/3}{x+1}\right)dx = x + \frac{5}{3}\ln|x-2| - \frac{2}{3}\ln|x+1| + C. \]
