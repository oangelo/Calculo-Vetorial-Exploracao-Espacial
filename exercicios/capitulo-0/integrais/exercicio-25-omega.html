• EXERCÍCIO 30 [NÍVEL OMEGA]
    ORIGEM: Desafio de Identificação - Seção de Análise Avançada [SIGMA-4]

    Este exercício não fornece dica. Você deve identificar qual técnica de integração aplicar.

    Calcule a integral:

    \[\int \frac{dx}{x\sqrt{x^2 - 1}}\]

    RELATÓRIO DE CÁLCULO [SIGMA-4 REQUERIDO]

    Substituição trigonométrica: seja \(x = \sec(\theta)\), então \(dx = \sec(\theta)\tan(\theta)\,d\theta\) e \(\sqrt{x^2 - 1} = \tan(\theta)\).

    \[\int \frac{\sec(\theta)\tan(\theta)\,d\theta}{\sec(\theta)\cdot\tan(\theta)} = \int d\theta = \theta + C.\]

    Como \(x = \sec(\theta)\), temos \(\theta = \arcsec(x)\). Portanto:

    \[\int \frac{dx}{x\sqrt{x^2 - 1}} = \arcsec(x) + C.\]

    Alternativa: a substituição \(u = \sqrt{x^2 - 1}\) também funciona, levando a \(\arctan(u) + C\), o que é equivalente pela identidade \(\arcsec(x) = \arctan(\sqrt{x^2 - 1})\) para \(x > 1\).
