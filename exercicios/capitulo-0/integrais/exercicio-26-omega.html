• EXERCÍCIO 31 [NÍVEL OMEGA]
    ORIGEM: Desafio de Identificação - Departamento de Cálculos Avançados [SIGMA-4]

    Este exercício não fornece dica. Você deve identificar qual técnica de integração aplicar.

    Calcule a integral:

    \[\int \ln^2(x)\,dx\]

    RELATÓRIO DE CÁLCULO [SIGMA-4 REQUERIDO]

    Esta integral exige integração por partes duas vezes. Escolha \(u = \ln^2(x)\) e \(dv = dx\). Então \(du = 2\ln(x)\cdot\frac{1}{x}\,dx\) e \(v = x\).

    Aplicando \(\int u\,dv = uv - \int v\,du\):

    \[\int \ln^2(x)\,dx = x\ln^2(x) - \int x\cdot\frac{2\ln(x)}{x}\,dx = x\ln^2(x) - 2\int \ln(x)\,dx.\]

    A integral restante \(\int \ln(x)\,dx\) resolve-se por partes novamente (ou é conhecida):

    \[\int \ln(x)\,dx = x\ln(x) - x + C.\]

    Substituindo:

    \[\int \ln^2(x)\,dx = x\ln^2(x) - 2x\ln(x) + 2x + C = x\bigl(\ln^2(x) - 2\ln(x) + 2\bigr) + C.\]
