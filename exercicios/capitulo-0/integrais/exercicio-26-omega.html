• EXERCÍCIO 26 [NÍVEL OMEGA]
    ORIGEM: Desafio de Identificação - Departamento de Cálculos Orbitais [SIGMA-4]

    Este exercício não fornece dica. Você deve identificar qual técnica de integração aplicar.

    Calcule a integral:

    \[\int \frac{1}{\cos^3(x)}\,dx\]

    RELATÓRIO DE CÁLCULO [SIGMA-4 REQUERIDO]

    Note que \(\frac{1}{\cos^3(x)} = \sec^3(x)\). Esta é a integral do Exercício 20 — mas sem dica, o aluno deve reconhecer a forma e lembrar (ou derivar) o método.

    Escreva \(\sec^3(x) = \sec(x)\cdot\sec^2(x)\) e aplique integração por partes com \(u = \sec(x)\), \(dv = \sec^2(x)\,dx\). Então \(du = \sec(x)\tan(x)\,dx\) e \(v = \tan(x)\).

    \[I = \sec(x)\tan(x) - \int \sec(x)\tan^2(x)\,dx = \sec(x)\tan(x) - \int \sec(x)(\sec^2(x) - 1)\,dx\]

    \[= \sec(x)\tan(x) - I + \int \sec(x)\,dx\]

    Resolvendo para \(I\): \[I = \frac{1}{2}\sec(x)\tan(x) + \frac{1}{2}\ln|\sec(x) + \tan(x)| + C.\]
