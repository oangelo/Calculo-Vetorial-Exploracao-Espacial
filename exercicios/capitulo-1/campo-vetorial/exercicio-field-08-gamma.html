• EXERCÍCIO VECTOR-08-GAMMA
    ORIGEM: Laboratório de Física Espacial - Projeto Ranger [DADOS DE 18/AGO/1961]

    CONEXÃO: Até agora, cada campo tinha UMA fonte. Mas na prática, temos MÚLTIPLAS fontes (dois planetas, duas cargas). O campo total é a SOMA dos campos individuais — este é o princípio de superposição. Vamos trabalhar com um dipolo elétrico.

    Considere o campo elétrico de duas cargas pontuais: \[\vec{E}(x,y) = \vec{E}_- + \vec{E}_+\] onde a carga negativa em \((-1,0)\) produz: \[\vec{E}_-(x,y) = \frac{k}{[(x+1)^2 + y^2]^{3/2}} \begin{pmatrix} -(x+1) \ -y \end{pmatrix}\] e a carga positiva em \((1,0)\) produz: \[\vec{E}_+(x,y) = \frac{k}{[(x-1)^2 + y^2]^{3/2}} \begin{pmatrix} (x-1) \ y \end{pmatrix}\]

    NOTA: o campo de uma carga aponta PARA a carga (se negativa) ou PARA LONGE da carga (se positiva), com intensidade proporcional a \(1/r^2\).

    (a) Antes de calcular: as duas cargas são simétricas em relação ao eixo \(y\). Onde você esperaria que o campo resultante seja nulo por simetria? Por quê?

    (b) Calcule \(\vec{E}_-\) e \(\vec{E}_+\) separadamente no ponto \((0,1)\). Use \(k=1\) para simplificar.

    (c) Some os dois para obter \(\vec{E}(0,1)\). Em que direção o campo resultante aponta?

    (d) Por simetria, o campo é nulo em algum ponto do eixo \(y\)? Explique.

    (e) Esboce o campo ao longo do eixo \(x\) para \(x > 1\). Para que lado aponta?

    ARQUIVO AUXILIAR [SIGMA-N]
    [NOTA TÉCNICA]: Para (b), calcule cada campo separadamente — não tente simplificar a fórmula completa de uma vez. Para (d), pense na simetria: o ponto \((0,y)\) está à mesma distância de ambas as cargas.
    RELATÓRIO DE CÁLCULO [NÍVEL 4 REQUERIDO]

    PROTOCOLO DE ANÁLISE VETORIAL:

    (a) PREVISÃO POR SIMETRIA:

    As duas cargas são simétricas em relação ao eixo \(y\). Na origem \((0,0)\), as contribuições das duas cargas se cancelam exatamente pela simetria do arranjo. É razoável esperar que o campo resultante seja nulo na origem.

    (b) CAMPOS SEPARADOS EM \((0,1)\):

    Para \(\vec{E}_-\) com a carga em \((-1,0)\): distância = \(\sqrt{(0+1)^2 + 1^2} = \sqrt{2}\), então \(\vec{E}_-(0,1) = \frac{1}{(\sqrt{2})^3} \begin{pmatrix} -1 \ -1 \end{pmatrix} = \frac{1}{2\sqrt{2}} \begin{pmatrix} -1 \ -1 \end{pmatrix}\).

    Para \(\vec{E}_+\) com a carga em \((1,0)\): distância = \(\sqrt{(0-1)^2 + 1^2} = \sqrt{2}\), então \(\vec{E}_+(0,1) = \frac{1}{2\sqrt{2}} \begin{pmatrix} -1, 1 \end{pmatrix}\).

    (c) CAMPO RESULTANTE:

    \[\vec{E}(0,1) = \frac{1}{2\sqrt{2}} \begin{pmatrix} -1 \ -1 \end{pmatrix} + \frac{1}{2\sqrt{2}} \begin{pmatrix} -1, 1 \end{pmatrix} = \frac{1}{2\sqrt{2}} \begin{pmatrix} -2, 0 \end{pmatrix}\] O campo resultante aponta para a ESQUERDA (sentido negativo do eixo \(x\)). As componentes \(y\) se cancelaram por simetria, restando apenas a componente horizontal.

    (d) SIMETRIA:

    Por simetria, o campo na origem \((0,0)\) é nulo: as contribuições das duas cargas se cancelam exatamente. Em qualquer ponto \((0,y)\) do eixo \(y\), as componentes \(x\) dos dois campos são iguais (por simetria), mas ambas apontam para a esquerda (atraídas pela carga negativa à esquerda e repelidas pela positiva à direita). Portanto, o campo só é nulo na origem.

    (e) CAMPO NO EIXO \(x\) PARA \(x > 1\):

    Ambos os campos apontam para a direita (a carga positiva repele, a negativa atrai), então o campo resultante aponta para a DIREITA. A intensidade diminui com a distância.
