• EXERCÍCIO VECTOR-01-ALFA
    ORIGEM: Departamento de Análise de Campos — Projeto Redstone [RELATÓRIO TÉCNICO CLASSIFICADO, 1953]

    Na análise de um campo vetorial, a divergência nos diz se o campo "se expande" ou "se comprime" em cada ponto. Antes de calcular, vamos aprender a enxergar essa propriedade.

    Considere o campo vetorial \[\vec{F}(x, y) = (x,\; y).\] Este campo aponta radialmente para fora a partir da origem — cada vetor aponta "afastando-se" do centro.

    (a) Desenhe (ou visualize mentalmente) o campo nos pontos \((1,0)\), \((0,1)\), \((-1,0)\) e \((0,-1)\). Em qual desses pontos o campo "jorra para fora"? Em qual ele "converge para dentro"?

    (b) Agora considere o campo \(\vec{G}(x,y) = (-y,\; x)\). Este campo gira em torno da origem (sentido anti-horário). Compare: em \(\vec{G}\), o campo se expande, se comprime, ou apenas gira? Justifique.

    (c) Com base nas suas observações, classifique intuitivamente qual desses dois campos deve ter divergência positiva na origem e qual deve ter divergência zero. (Não calcule ainda — apenas argumente.)

    ARQUIVO AUXILIAR [SIGMA-N]
    [NOTA TÉCNICA]: Para desenhar o campo \(\vec{F}(x,y) = (x,y)\) no ponto \((1,0)\), o vetor é \((1,0)\) — uma seta apontando para a direita. No ponto \((0,1)\), o vetor é \((0,1)\) — uma seta para cima. O padrão é: todas as setas apontam para longe da origem.
    RELATÓRIO DE CÁLCULO [SIGMA-3 REQUERIDO]

    (a) Visualização de \(\vec{F} = (x,y)\):

    No ponto \((1,0)\): vetor \((1,0)\) — aponta para a direita (para fora). No ponto \((0,1)\): vetor \((0,1)\) — aponta para cima (para fora). No ponto \((-1,0)\): vetor \((-1,0)\) — aponta para a esquerda (para fora). No ponto \((0,-1)\): vetor \((0,-1)\) — aponta para baixo (para fora).

    Em todos os pontos, o campo aponta para fora — há uma "fonte" na origem.

    (b) Comparação com \(\vec{G} = (-y, x)\):

    No ponto \((1,0)\): vetor \((0,1)\) — para cima. No ponto \((0,1)\): vetor \((-1,0)\) — para a esquerda. O campo gira em torno da origem sem se expandir nem comprimir. Nenhum ponto parece ser fonte ou sumidouro.

    (c) Classificação intuitiva:

    \(\vec{F} = (x,y)\) deve ter divergência positiva (expansão). \(\vec{G} = (-y,x)\) deve ter divergência zero (apenas rotação, sem expansão).

    Verificação (adiantamento): \(\nabla \cdot \vec{F} = \frac{\partial x}{\partial x} + \frac{\partial y}{\partial y} = 1 + 1 = 2 > 0\). Correto — campo expansivo. \(\nabla \cdot \vec{G} = \frac{\partial(-y)}{\partial x} + \frac{\partial x}{\partial y} = 0 + 0 = 0\). Correto — campo sem expansão.
