• EXERCÍCIO VECTOR-04-BETA
    ORIGEM: Seção de Análises Orbitais — Programa Apollo [DOCUMENTO PARCIALMENTE CENSURADO]

    Já calculamos divergências de vários campos. Agora vamos examinar um conceito importante: quando um campo é chamado de solenoidal.

    Um campo \\(\\vec{F}\\) é solenoidal se \\(\\nabla \\cdot \\vec{F} = 0\\) em todo o espaço. Isso significa que o campo não tem fontes nem sumidouros — o que "entra" em qualquer região é igual ao que "sai".

    Considere o campo \\(\\vec{M}(x,y,z) = (x^2 - y^2,\\; xy,\\; x + y + z)\\).

    • (a) Calcule \\(\\nabla \\cdot \\vec{M}\\). O resultado é identicamente zero?
    • (b) Se o campo não é solenoidal, encontre as regiões onde \\(\\nabla \\cdot \\vec{M} = 0\\). (Dica: a equação depende apenas de uma variável.)
    • (c) Cuidado com a armadilha conceitual: Um campo com divergência zero (solenoidal) não é o mesmo que um campo sem rotação (irrotacional). O rotacional \\(\\nabla \\times \\vec{F}\\) é uma informação diferente. Portanto, saber apenas o valor da divergência não permite concluir se o campo é irrotacional. Justifique por que são propriedades independentes.
    ARQUIVO AUXILIAR [SIGMA-N]
    [NOTA TÉCNICA]: Para \\(\\frac{\\partial}{\\partial x}(x^2 - y^2)\\), \\(y^2\\) é constante. Para \\(\\frac{\\partial}{\\partial y}(xy)\\), \\(x\\) é constante, e a derivada de \\(y\\) é \\(1\\), então o resultado é \\(x\\).
    RELATÓRIO DE CÁLCULO [SIGMA-3 REQUERIDO]

    (a) Cálculo da divergência:

    \\[ \\nabla \\cdot \\vec{M} = \\frac{\\partial}{\\partial x}(x^2 - y^2) + \\frac{\\partial}{\\partial y}(xy) + \\frac{\\partial}{\\partial z}(x + y + z) \\] \\[ = 2x + x + 1 = 3x + 1 \\]

    O resultado \\(3x + 1\\) não é identicamente zero. Logo, o campo não é solenoidal.

    (b) Região onde a divergência se anula:

    \\(3x + 1 = 0 \\implies x = -\\frac{1}{3}\\). O campo tem divergência nula apenas no plano \\(x = -1/3\\).

    (c) Propriedades independentes:

    Um campo pode ser solenoidal mas não irrotacional (ex: \\((-y, x, 0)\\) — divergência zero, mas gira). Pode ser irrotacional mas não solenoidal (ex: \\((x, 0, 0)\\) — rotacional zero, mas divergência 1). A divergência mede expansão; o rotacional mede rotação. São "direções" diferentes de análise do campo.
