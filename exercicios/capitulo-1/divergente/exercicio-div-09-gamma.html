• EXERCÍCIO VECTOR-09-GAMMA
    ORIGEM: Análise Dinâmica de Fluidos — Projeto Skylab [RELATÓRIO TÉCNICO, 1973]

    Já vimos campos com divergência positiva, negativa, constante e zero. Agora vamos combinar: um campo onde a divergência pode mudar de sinal dependendo do ponto.

    Considere o campo \[\vec{F}(x,y,z) = (x^2 z,\; -2xyz,\; xy^2)\].

    (a) Calcule cada termo da divergência separadamente. Note que o campo tem uma "simetria" entre os termos.

    (b) Simplifique a soma total. O resultado depende de quais variáveis?

    (c) Classifique as regiões do espaço: onde a divergência é positiva, onde é negativa, e onde é zero? (Dica: analise o sinal do resultado como função de \(x\) e \(z\).)

    ARQUIVO AUXILIAR [SIGMA-N]
    [NOTA TÉCNICA]: \(\frac{\partial}{\partial x}(x^2 z) = 2xz\), \(\frac{\partial}{\partial y}(-2xyz) = -2xz\), e \(\frac{\partial}{\partial z}(xy^2) = 0\). Note a cancelação entre os dois primeiros termos!
    RELATÓRIO DE CÁLCULO [SIGMA-3 REQUERIDO]

    (a) Termos individuais:

    \[ \frac{\partial}{\partial x}(x^2 z) = 2xz \] \[ \frac{\partial}{\partial y}(-2xyz) = -2xz \] \[ \frac{\partial}{\partial z}(xy^2) = 0 \]

    (b) Divergência total:

    \[ \nabla \cdot \vec{F} = 2xz - 2xz + 0 = 0 \]

    O resultado é zero em todo o espaço! Este campo é solenoidal.

    (c) Análise:

    Com divergência identicamente zero, o campo não possui nem fontes nem sumidouros em lugar algum. O que "entra" em qualquer região fechada é exatamente o que "sai" pela fronteira. Note como o termo \(x^2 z\) no primeiro componente e \(-2xyz\) no segundo se "compensam" perfeitamente — uma construção intencional de campo solenoidal.
