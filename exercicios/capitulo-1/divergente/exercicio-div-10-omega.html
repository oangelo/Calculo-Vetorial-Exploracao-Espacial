• EXERCÍCIO VECTOR-10-OMEGA
    ORIGEM: Laboratório de Análise Avançada — Projeto Mercury [DOCUMENTO ULTRASECRETO, 1958]

    Nos exercícios anteriores, calculamos divergências de campos dados. Agora vamos ao inverso: construir um campo com uma propriedade desejada. Este tipo de problema exige criatividade — não existe fórmula, apenas raciocínio.

    Encontre um campo vetorial \\(\\vec{F}(x,y,z)\\) que satisfaça ambas as condições: \\[\\nabla \\cdot \\vec{F} = 3 \\quad \\text{(divergência constante)}\\] \\[\\nabla \\times \\vec{F} = \\vec{0} \\quad \\text{(rotacional nulo)}\\]

    • (a) Dica para o início: se o rotacional é nulo, o campo não pode ter componentes "cruzadas" como \\((-y, x, 0)\\). Tente um campo da forma \\(\\vec{F} = (f(x,y,z),\\; g(x,y,z),\\; h(x,y,z))\\) onde cada componente depende apenas de "sua" variável. Por exemplo: \\(\\vec{F} = (ax,\\; by,\\; cz)\\). Calcule a divergência e o rotacional deste campo.
    • (b) Que valores de \\(a\\), \\(b\\) e \\(c\\) satisfazem as duas condições?
    • (c) Verifique: \\(\\nabla \\times \\vec{F} = 0\\) significa que as "derivadas cruzadas" se igualam. Confirme explicitamente que seu campo satisfaz essa condição.
    RELATÓRIO DE CÁLCULO [SIGMA-3 REQUERIDO]

    (a) Tentativa:

    Seja \\(\\vec{F} = (ax,\\; by,\\; cz)\\).

    \\[ \\nabla \\cdot \\vec{F} = a + b + c \\] \\[ \\nabla \\times \\vec{F} = \\left(\\frac{\\partial h}{\\partial y} - \\frac{\\partial g}{\\partial z},\\; \\frac{\\partial f}{\\partial z} - \\frac{\\partial h}{\\partial x},\\; \\frac{\\partial g}{\\partial x} - \\frac{\\partial f}{\\partial y}\\right) = (0 - 0,\\; 0 - 0,\\; 0 - 0) = \\vec{0} \\]

    O rotacional é zero automaticamente, pois cada componente depende apenas de sua própria variável.

    (b) Valores de \\(a\\), \\(b\\), \\(c\\):

    Precisamos de \\(a + b + c = 3\\). A escolha mais simples: \\(a = b = c = 1\\), ou seja, \\[\\vec{F} = \\frac{3}{3}(x,\\; y,\\; z) = (x,\\; y,\\; z)\\]

    (c) Verificação do rotacional:

    \\[ \\nabla \\times (x, y, z) = \\left(\\frac{\\partial z}{\\partial y} - \\frac{\\partial y}{\\partial z},\\; \\frac{\\partial x}{\\partial z} - \\frac{\\partial z}{\\partial x},\\; \\frac{\\partial y}{\\partial x} - \\frac{\\partial x}{\\partial y}\\right) = (0, 0, 0) \\quad \\checkmark \\]

    \\[ \\nabla \\cdot (x, y, z) = 1 + 1 + 1 = 3 \\quad \\checkmark \\]

    O campo \\(\\vec{F}(x,y,z) = (x, y, z)\\) satisfaz ambas as condições.
