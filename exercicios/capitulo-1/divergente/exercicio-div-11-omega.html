• EXERCÍCIO VECTOR-11-OMEGA
    ORIGEM: Seção de Teoria de Campos — Pesquisa Fundamental [DOCUMENTO ULTRASECRETO, 1960]

    Ao longo desta lista, trabalhamos com divergência e, em momentos anteriores, com rotacional. Agora vamos conectar os dois conceitos de forma rigorosa, provando uma identidade vetorial fundamental.

    A identidade \(\nabla \times (\nabla f) = \vec{0}\) diz que o rotacional de um gradiente é sempre nulo. Geometricamente: um campo que aponta na "direção de maior crescimento" de uma função não pode ter rotação.

    (a) Calcule o gradiente de \(f(x,y,z) = x^2 y + yz^3\). Ou seja, encontre \(\nabla f = \left(\frac{\partial f}{\partial x},\; \frac{\partial f}{\partial y},\; \frac{\partial f}{\partial z}\right)\).

    (b) Agora calcule o rotacional \(\nabla \times (\nabla f)\). Lembre-se: para \(\vec{G} = (G_1, G_2, G_3)\), \[\nabla \times \vec{G} = \left(\frac{\partial G_3}{\partial y} - \frac{\partial G_2}{\partial z},\; \frac{\partial G_1}{\partial z} - \frac{\partial G_3}{\partial x},\; \frac{\partial G_2}{\partial x} - \frac{\partial G_1}{\partial y}\right)\] Substitua \(\vec{G} = \nabla f\) e verifique que cada componente do rotacional é zero.

    (c) Observe que, no cálculo, aparecem "derivadas mistas" como \(\frac{\partial^2 f}{\partial y \partial x}\) e \(\frac{\partial^2 f}{\partial x \partial y}\). Pelo Teorema de Clairaut (ou "igualdade das derivadas mistas"), se \(f\) tem derivadas parciais contínuas, então \(\frac{\partial^2 f}{\partial x \partial y} = \frac{\partial^2 f} {\partial y \partial x}\). É isso que faz cada componente do rotacional se anular. Explique como cada termo do rotacional envolve um par de derivadas mistas que se cancelam.

    RELATÓRIO DE CÁLCULO [SIGMA-4 REQUERIDO]

    (a) Gradiente:

    \[\frac{\partial f}{\partial x} = 2xy\] \[\frac{\partial f}{\partial y} = x^2 + z^3\] \[\frac{\partial f}{\partial z} = 3yz^2\]

    \[\nabla f = (2xy,\; x^2 + z^3,\; 3yz^2)\]

    (b) Rotacional do gradiente:

    Seja \(\vec{G} = \nabla f = (2xy, x^2 + z^3, 3yz^2)\).

    Componente \(\hat{i}\): \(\frac{\partial G_3}{\partial y} - \frac{\partial G_2}{\partial z} = \frac{\partial(3yz^2)}{\partial y} - \frac{\partial(x^2+z^3)}{\partial z} = 3z^2 - 3z^2 = 0\)

    Componente \(\hat{j}\): \(\frac{\partial G_1}{\partial z} - \frac{\partial G_3}{\partial x} = \frac{\partial(2xy)}{\partial z} - \frac{\partial(3yz^2)}{\partial x} = 0 - 0 = 0\)

    Componente \(\hat{k}\): \(\frac{\partial G_2}{\partial x} - \frac{\partial G_1}{\partial y} = \frac{\partial(x^2+z^3)}{\partial x} - \frac{\partial(2xy)}{\partial y} = 2x - 2x = 0\)

    \[\nabla \times (\nabla f) = (0, 0, 0) = \vec{0} \quad \checkmark\]

    (c) Derivadas mistas e Clairaut:

    Cada componente do rotacional envolve derivadas de componentes diferentes do gradiente em variáveis diferentes:

    Componente \(\hat{i}\): \(\frac{\partial}{\partial y}\left( \frac{\partial f}{\partial z}\right) - \frac{\partial}{\partial z} \left(\frac{\partial f}{\partial y}\right) = \frac{\partial^2 f}{\partial y \partial z} - \frac{\partial^2 f} {\partial z \partial y} = 0\) (Clairaut).

    Componente \(\hat{j}\): \(\frac{\partial}{\partial z}\left( \frac{\partial f}{\partial x}\right) - \frac{\partial}{\partial x} \left(\frac{\partial f}{\partial z}\right) = \frac{\partial^2 f}{\partial z \partial x} - \frac{\partial^2 f} {\partial x \partial z} = 0\) (Clairaut).

    Componente \(\hat{k}\): \(\frac{\partial}{\partial x}\left( \frac{\partial f}{\partial y}\right) - \frac{\partial}{\partial y} \left(\frac{\partial f}{\partial x}\right) = \frac{\partial^2 f}{\partial x \partial y} - \frac{\partial^2 f} {\partial y \partial x} = 0\) (Clairaut).

    A identidade \(\nabla \times (\nabla f) = \vec{0}\) é uma consequência direta da simetria das derivadas mistas. Qualquer campo que seja o gradiente de uma função escalar é irrotacional — não possui rotação.
