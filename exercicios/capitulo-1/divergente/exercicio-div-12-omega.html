• EXERCÍCIO VECTOR-12-OMEGA
    ORIGEM: Seção de Teoria de Campos — Pesquisa Fundamental [DOCUMENTO ULTRASECRETO, 1960]

    Para encerrar esta lista, um problema de construção que exige todo o conhecimento acumulado. Você deve criar um campo com propriedades específicas — uma habilidade essencial para engenheiros e físicos.

    Encontre um campo vetorial \(\vec{F}\) com divergência constante \(c\) e rotacional zero. Diferente do Exercício 10 (onde \(c = 3\) e a resposta era \((x,y,z)\)), agora queremos a solução geral para qualquer \(c\).

    • (a) Generalize o raciocínio do Exercício 10. Se \(\vec{F} = (ax,\; by,\; cz)\), a divergência é \(a + b + c\) e o rotacional é zero. Como escolher \(a\), \(b\), \(c\) de modo que \(a + b + c = c\) (constante desejada)?
    • (b) Existem múltiplas respostas? Justifique. (Dica: a escolha \(a = b = c/3\) funciona, mas é a única?)
    • (c) Verifique explicitamente que seu campo satisfaz ambas as condições.
    RELATÓRIO DE CÁLCULO [SIGMA-3 REQUERIDO]

    (a) Generalização:

    Queremos \(a + b + c = c\) (constante) e rotacional zero. A escolha mais natural é \(a = b = c/3\) (usando \(c\) como constante, renomeando: \(a = b = k = c/3\)).

    \[ \vec{F} = \frac{c}{3}(x,\; y,\; z) \]

    \[ \nabla \cdot \vec{F} = \frac{c}{3}(1 + 1 + 1) = c \quad \checkmark \]

    \[ \nabla \times \vec{F} = \vec{0} \quad \checkmark \]

    (b) Unicidade:

    Não é a única resposta. Qualquer campo da forma \((ax, by, cz)\) com \(a + b + c = c\) funciona. Exemplo: \(\vec{F} = (c \cdot x, 0, 0)\) também satisfaz. A escolha simétrica \(a = b = c/3\) é a mais elegante por isotropia — o campo "cresce" igualmente em todas as direções.

    (c) Verificação geral:

    Para \(\vec{F} = \frac{c}{3}(x, y, z)\): o rotacional é zero pois cada componente depende apenas de sua variável, e a divergência é \(\frac{c}{3}(1 + 1 + 1) = c\), como desejado.
