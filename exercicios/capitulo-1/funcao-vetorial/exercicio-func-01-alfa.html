• EXERCÍCIO VECTOR-01-ALFA
    ARQUIVO: Programa Vostok (1961) — [DADOS PARCIALMENTE DESCLASSIFICADOS]

    A trajetória de uma cápsula espacial em órbita baixa pode ser descrita por uma função vetorial no plano: \[ \vec{r}(t) = \bigl(R\cos t,\; R\sin t\bigr), \] onde \(R\) é o raio orbital (aproximadamente constante) e \(t\) é o tempo (em segundos). Note que, diferentemente das equações paramétricas de reta que você viu no capítulo anterior (onde \(x\) e \(y\) eram funções lineares de \(t\)), aqui as componentes são trigonométricas — o que gera uma curva fechada em vez de uma reta.

    (a) Identifique as componentes \(x(t)\) e \(y(t)\) de \(\vec{r}(t)\).

    (b) Calcule \(\|\vec{r}(t)\|\) usando a identidade trigonométrica \(\cos^2 t + \sin^2 t = 1\) (revisada em Cônicas, Cap 0). O módulo é constante ou varia com \(t\)?

    (c) Explique por que essa função descreve um movimento circular e não uma reta, como no caso das equações paramétricas \(\vec{r}(t) = P_0 + t\vec{v}\).

    ARQUIVO AUXILIAR [SIGMA-1]
    [DICA OPERACIONAL]: Para o item (b), lembre que o módulo de um vetor \((a, b)\) é \(\sqrt{a^2 + b^2}\). Substitua as componentes e use \(\cos^2 t + \sin^2 t = 1\). Para o item (c), compare com o que acontece com \(\|\vec{r}(t)\|\) quando \(\vec{r}(t) = P_0 + t\vec{v}\) — no caso da reta, o módulo geralmente varia com \(t\).
    RELATÓRIO DE CÁLCULO [SIGMA-1 REQUERIDO]

    SOLUÇÃO:

    (a) As componentes são \(x(t) = R\cos t\) e \(y(t) = R\sin t\).

    (b) \(\|\vec{r}(t)\| = \sqrt{(R\cos t)^2 + (R\sin t)^2} = \sqrt{R^2(\cos^2 t + \sin^2 t)} = \sqrt{R^2} = R\). O módulo é constante — igual a \(R\) para todo \(t\).

    (c) Como \(\|\vec{r}(t)\| = R\) para todo \(t\), o ponto \(\bigl(x(t), y(t)\bigr)\) permanece sempre à distância \(R\) da origem — isso define uma circunferência de raio \(R\). Já numa reta \(\vec{r}(t) = P_0 + t\vec{v}\), a distância à origem cresce (ou decresce) com \(t\), pois o objeto se afasta indefinidamente. A função vetorial com componentes trigonométricas cria um movimento periódico e fechado, diferente do movimento retilíneo das equações paramétricas de reta.
