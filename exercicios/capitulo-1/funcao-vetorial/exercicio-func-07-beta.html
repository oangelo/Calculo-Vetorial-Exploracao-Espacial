• EXERCÍCIO VECTOR-07-BETA
    ARQUIVO: Projeto Skylab (1973) — [DADOS PARCIALMENTE DESCLASSIFICADOS]

    Nos exercícios anteriores, você calculou velocidades e acelerações de funções vetoriais separadamente. Agora vamos fazer uma análise completa de uma trajetória: posição, velocidade, aceleração e comportamento do módulo — tudo junto. Esse é o tipo de análise que engenheiros de missão fazem para prever a trajetória de uma cápsula. Para derivar os produtos como \(e^t \cos t\), use a regra do produto de Cap 0: \(\frac{d}{dt}[f(t) \cdot g(t)] = f'(t)g(t) + f(t)g'(t)\).

    O movimento descrito por: \[ \vec{r}(t) = \bigl(e^t \cos t,\; e^t \sin t\bigr) \] simula combinações de rotação e expansão em condições de gravidade reduzida.

    (a) Calcule \(\|\vec{r}(t)\|\). O módulo é constante (círculo) ou varia (espiral)? Justifique.

    (b) Calcule o vetor velocidade \(\vec{r}'(t)\).

    (c) Calcule o vetor aceleração \(\vec{r}''(t)\).

    (d) Analise o comportamento de \(\|\vec{r}(t)\|\) quando \(t \to +\infty\). O objeto se afasta ou se aproxima da origem?

    ARQUIVO AUXILIAR [SIGMA-7]
    [DICA OPERACIONAL]: Para o item (a), lembre da identidade trigonométrica \(\cos^2 t + \sin^2 t = 1\) (Cap 0 — Cônicas). Para derivar \(e^t \cos t\), aplique a regra do produto: \(\frac{d}{dt}[e^t \cos t] = e^t \cos t + e^t(-\sin t) = e^t(\cos t - \sin t)\). O fator \(e^t\) é comum a ambas as componentes — extraia-o para simplificar.
    RELATÓRIO DE CÁLCULO [SIGMA-7 REQUERIDO]

    SOLUÇÃO:

    (a) \(\|\vec{r}(t)\| = \sqrt{(e^t \cos t)^2 + (e^t \sin t)^2} = \sqrt{e^{2t}(\cos^2 t + \sin^2 t)} = e^t\). O módulo NÃO é constante — cresce exponencialmente. Portanto, a trajetória é uma espiral (não um círculo).

    (b) Aplicando a regra do produto a cada componente: \[ \vec{r}'(t) = \bigl(e^t(\cos t - \sin t),\; e^t(\sin t + \cos t)\bigr). \]

    (c) Derivando novamente: \[ \vec{r}''(t) = \bigl(-2e^t \sin t,\; 2e^t \cos t\bigr). \]

    (d) Como \(\|\vec{r}(t)\| = e^t\), quando \(t \to +\infty\), o módulo tende ao infinito. O objeto se afasta exponencialmente da origem enquanto gira — um movimento espiral expansivo. Em um contexto físico, isso poderia representar uma sonda escapando de um campo gravitacional fraco.
