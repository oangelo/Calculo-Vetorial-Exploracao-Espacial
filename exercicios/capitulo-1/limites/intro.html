PRÉ-REQUISITOS

Conceitos necessários para esta lista de exercícios:

  • Funções elementares: polinômios, funções trigonométricas (\(\sin\), \(\cos\)), exponencial (\(e^x\)), logaritmo (\(\ln\)), raiz quadrada. Você deve estar confortável com o comportamento de cada uma.
  • Limites em uma variável (Cálculo 1): definição de \(\lim_{t o a} f(t)\), cálculo por substituição direta, limites laterais (\(t o a^+\), \(t o a^-\)), limites no infinito. Se você sabe que \(\lim_{t o 0} rac{\sin t}{t} = 1\) e \(\lim_{t o \infty} rac{1}{t} = 0\), está pronto.
  • Desigualdades fundamentais: \(|\sin x| \leq |x|\), \(|\cos x| \leq 1\), desigualdade AM-GM (\(a + b \geq 2\sqrt{ab}\) para \(a, b \geq 0\)). Estas são usadas no método do sanduíche.
  • Funções vetoriais (tópico anterior): \(\vec{r}(t)\) como trajetória de uma partícula, componentes contínuas, domínio de funções vetoriais. Este tópico é a conexão direta com limites.
  • Continuidade de funções de uma variável: uma função \(f(t)\) é contínua em \(t = a\) se \(\lim_{t o a} f(t) = f(a)\). Este conceito se generaliza para funções de duas variáveis.
  • Coordenadas polares (conceitual): \(x = r\cos heta\), \(y = r\sin heta\). Não é preciso dominar integração em polares — apenas a mudança de coordenadas.
  • Álgebra de frações: simplificação, fatoração, frações parciais básicas.
Se você se sente inseguro em algum destes pontos, recomendamos revisar a lista de exercícios do Capítulo 0 (Revisão), em especial a seção de Derivadas e Limites, antes de prosseguir.
