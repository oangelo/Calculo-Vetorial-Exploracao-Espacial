• EXERCÍCIO VECTOR-REVIEW-11
    ORIGEM: Prova de Identidade Vetorial Clássica — Após exercícios focados em cálculos específicos, agora provamos uma propriedade geral: a divergência de um rotacional é sempre zero. Esta identidade \(\nabla\cdot(\nabla\times\vec{A}) = 0\) é uma das mais importantes do cálculo vetorial. Ela diz que um campo rotacional nunca pode ter fontes — uma propriedade que será crucial para campos conservativos em capítulos futuros.

    Considere o campo \[\vec{A}(x,y,z)=\begin{pmatrix}yz\\xz\\xy\end{pmatrix}.\]

    (a) Calcule \(\nabla\times\vec{A}\) explicitamente.

    (b) Calcule \(\nabla\cdot(\nabla\times\vec{A})\) para este campo específico e verifique que o resultado é zero.

    (c) Demonstre analiticamente que \(\nabla\cdot(\nabla\times\vec{A})=0\) para qualquer campo suave \(\vec{A}\), expandindo as derivadas parciais e mostrando o cancelamento termo a termo.

    ARQUIVO AUXILIAR [SIGMA-4]
    Para (c), escreva \(\vec{A} = (P, Q, R)\) e expanda o rotacional em componentes. A divergência do resultado envolve derivadas segundas mistas: \(\partial^2 R/\partial y\partial x\) e \(\partial^2 Q/\partial x\partial y\). Pela igualdade das derivadas parciais mistas (Clairaut), elas se cancelam.
    RELATÓRIO DE CÁLCULO [SIGMA-4 REQUERIDO]

    (a) \[\nabla\times\vec{A} = \begin{pmatrix} \partial_y(xy) - \partial_z(xz) \\ \partial_z(yz) - \partial_x(xy) \\ \partial_x(xz) - \partial_y(yz) \end{pmatrix} = \begin{pmatrix} x - x \\ y - y \\ z - z \end{pmatrix} = \vec{0}.\]

    Para este campo específico, o rotacional já é zero, então \(\nabla\cdot\vec{0} = 0\) trivialmente.

    (b) Verificação completa usando \(\vec{B} = \nabla\times\vec{A} = (0, 0, 0)\): \[\nabla\cdot\vec{B} = 0.\]

    (c) Prova geral para \(\vec{A} = (P, Q, R)\):

    \[\nabla\times\vec{A} = \left(\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z}, \frac{\partial P}{\partial z} - \frac{\partial R}{\partial x}, \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right)\]

    \[\nabla\cdot(\nabla\times\vec{A}) = \frac{\partial}{\partial x}\left(\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z}\right) + \frac{\partial}{\partial y}\left(\frac{\partial P}{\partial z} - \frac{\partial R}{\partial x}\right) + \frac{\partial}{\partial z}\left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right)\]

    \[= \frac{\partial^2 R}{\partial x\partial y} - \frac{\partial^2 Q}{\partial x\partial z} + \frac{\partial^2 P}{\partial y\partial z} - \frac{\partial^2 R}{\partial y\partial x} + \frac{\partial^2 Q}{\partial z\partial x} - \frac{\partial^2 P}{\partial z\partial y}\]

    Pela igualdade das derivadas parciais mistas (teorema de Clairaut), cada par se cancela: \(\partial^2 R/\partial x\partial y = \partial^2 R/\partial y\partial x\), etc. Logo o resultado é \(0\). \(\blacksquare\)
