• EXERCÍCIO VECTOR-REVIEW-1
    ORIGEM: Revisão Geral [Programa Apollo, 1969] — Iniciamos esta lista de revisão retomando os fundamentos de geometria analítica do Cap 0. Todo cálculo com vetores começa aqui: primeiro dominamos a norma, depois avançamos para produtos.

    Considere o vetor \[\vec{v} = (3, -4, 12).\]

    (a) Calcule a magnitude \(|\vec{v}|\).

    (b) Determine o vetor unitário \(\hat{v} = \vec{v}/|\vec{v}|\) na mesma direção.

    (c) Verifique que \(|\hat{v}| = 1\), confirmando que a normalização está correta.

    ARQUIVO AUXILIAR [SIGMA-9]
    [NOTA TÉCNICA]: Use \(|\vec{v}| = \sqrt{v_1^2 + v_2^2 + v_3^2}\). Para o vetor unitário, divida cada componente pela magnitude.
    RELATÓRIO DE CÁLCULO [NÍVEL 4 REQUERIDO]

    SOLUÇÃO:

    (a) \[|\vec{v}| = \sqrt{9 + 16 + 144} = \sqrt{169} = 13\]

    (b) \[\hat{v} = \frac{1}{13}(3, -4, 12) = \left(\frac{3}{13}, -\frac{4}{13}, \frac{12}{13}\right)\]

    (c) \[|\hat{v}| = \sqrt{\frac{9}{169} + \frac{16}{169} + \frac{144}{169}} = \sqrt{\frac{169}{169}} = 1 \quad \checkmark\]
