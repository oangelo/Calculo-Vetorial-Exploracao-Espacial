• EXERCÍCIO VECTOR-02-ALFA
    ORIGEM: Análise de Campos Magnéticos - Sistema de Navegação da Apollo

    No exercício anterior, você viu que um campo circular \((-y, x, 0)\) tem rotacional não-zero na direção \(z\). Agora vamos ver um campo onde a rotação acontece em uma direção diferente. O campo vetorial magnético \(\vec{B} = \begin{pmatrix} 0 \ -z \ y \end{pmatrix}\) modela um componente do campo magnético terrestre.

    Solicitação:
    • (a) Identifique as componentes \(P\), \(Q\) e \(R\) do campo \(\vec{B}\).
    • (b) Calcule a componente \(i\) do rotacional: \(\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z}\).
    • (c) Calcule a componente \(j\): \(\frac{\partial P}{\partial z} - \frac{\partial R}{\partial x}\).
    • (d) Calcule a componente \(k\): \(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\).
    • (e) O rotacional é zero? O campo é irrotacional? Em qual direção aponta a rotação?
    ARQUIVO AUXILIAR [SIGMA-N]
    [NOTA TÉCNICA]: Para \(\vec{B} = (0, -z, y)\), lembre-se que \(\frac{\partial y}{\partial y} = 1\) e \(\frac{\partial (-z)}{\partial z} = -1\). As derivadas parciais que você já domina são os blocos de construção do rotacional.
    RELATÓRIO DE CÁLCULO [NÍVEL 4 REQUERIDO]

    SOLUÇÃO:

    (a) \(P = 0\), \(Q = -z\), \(R = y\).

    (b) Componente \(i\): \(\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z} = \frac{\partial y}{\partial y} - \frac{\partial (-z)}{\partial z} = 1 - (-1) = 2\).

    (c) Componente \(j\): \(\frac{\partial P}{\partial z} - \frac{\partial R}{\partial x} = 0 - 0 = 0\).

    (d) Componente \(k\): \(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} = 0 - 0 = 0\).

    (e) \(\nabla \times \vec{B} = (2, 0, 0)\). O rotacional NÃO é zero — o campo não é irrotacional. A rotação aponta na direção \(+x\), diferente do exercício anterior onde a rotação era na direção \(+z\).
