• EXERCÍCIO VECTOR-03-ALFA
    ORIGEM: Fluxo de Plasma no Espaço Próximo à Lua

    Nos dois exercícios anteriores, os campos tinham rotacional não-zero — havia rotação local. Nem todo campo é assim. O campo vetorial \(\vec{E} = (yz, xz, xy)\) descreve um fluxo elétrico variável. Antes de calcular, observe o campo: as componentes são produtos de duas variáveis, com uma simetria particular.

    (a) Identifique \(P\), \(Q\) e \(R\). Note a simetria: cada componente é o produto das outras duas variáveis.

    (b) Calcule a componente \(i\) do rotacional: \(\frac{\partial (xy)}{\partial y} - \frac{\partial (xz)}{\partial z}\).

    (c) Calcule a componente \(j\) e a componente \(k\).

    (d) O rotacional é zero? O campo é irrotacional? O que isso significa fisicamente?

    ARQUIVO AUXILIAR [SIGMA-N]
    [NOTA TÉCNICA]: A simetria do campo faz com que as derivadas parciais se cancelem. Quando \(\frac{\partial R}{\partial y} = \frac{\partial Q}{\partial z}\), a componente \(i\) do rotacional é zero.
    RELATÓRIO DE CÁLCULO [NÍVEL 4 REQUERIDO]

    SOLUÇÃO:

    (a) \(P = yz\), \(Q = xz\), \(R = xy\).

    (b) Componente \(i\): \(\frac{\partial (xy)}{\partial y} - \frac{\partial (xz)}{\partial z} = x - x = 0\).

    (c) Componente \(j\): \(\frac{\partial (yz)}{\partial z} - \frac{\partial (xy)}{\partial x} = y - y = 0\). Componente \(k\): \(\frac{\partial (xz)}{\partial x} - \frac{\partial (yz)}{\partial y} = z - z = 0\).

    (d) \(\nabla \times \vec{E} = (0, 0, 0)\). O campo é irrotacional — não há rotação local em nenhum ponto. Isso significa que, localmente, o fluxo não gira.
