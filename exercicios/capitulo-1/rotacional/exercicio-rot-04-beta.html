• EXERCÍCIO VECTOR-04-BETA
    ORIGEM: Forças Aerodinâmicas em Retorno Orbital

    No exercício anterior, você viu um campo irrotacional onde as derivadas parciais se cancelavam por simetria. Agora vamos ver outro tipo de campo irrotacional — um onde cada componente depende apenas de uma variável. O campo \(\vec{V} = (x^2, y^2, z^2)\) representa a distribuição de velocidades de reentrada.

    Solicitação:
    • (a) Note que \(P = x^2\) depende apenas de \(x\), \(Q = y^2\) apenas de \(y\), e \(R = z^2\) apenas de \(z\). Calcule \(\frac{\partial R}{\partial y}\) e \(\frac{\partial Q}{\partial z}\). O que acontece?
    • (b) Calcule as outras duas componentes do rotacional.
    • (c) Todas são zero. Interprete: este campo não tem rotação local. Mas o fluxo é "reto"? Pense no que o campo faz no ponto \((1,1,1)\): o vetor é \((1,1,1)\), apontando na diagonal. O campo não é uniforme, mas não gira.
    ARQUIVO AUXILIAR [SIGMA-N]
    [NOTA TÉCNICA]: Quando cada componente depende apenas de sua própria variável, as derivadas cruzadas são sempre zero — e o rotacional se anula. Isso é diferente da simetria do exercício anterior.
    RELATÓRIO DE CÁLCULO [NÍVEL 4 REQUERIDO]

    SOLUÇÃO:

    (a) \(\frac{\partial R}{\partial y} = \frac{\partial (z^2)}{\partial y} = 0\) e \(\frac{\partial Q}{\partial z} = \frac{\partial (y^2)}{\partial z} = 0\). Componente \(i = 0 - 0 = 0\).

    (b) Componente \(j\): \(\frac{\partial P}{\partial z} - \frac{\partial R}{\partial x} = 0 - 0 = 0\). Componente \(k\): \(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} = 0 - 0 = 0\).

    (c) \(\nabla \times \vec{V} = (0, 0, 0)\). O campo não gira localmente. No ponto \((1,1,1)\), o vetor \((1,1,1)\) aponta para fora — é um fluxo de expansão, não de rotação. O rotacional mede rotação, não expansão (essa é a função do divergente, que veremos no próximo tópico).
