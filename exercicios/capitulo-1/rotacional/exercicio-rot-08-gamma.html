• EXERCÍCIO VECTOR-08-GAMMA
    ORIGEM: Análise de Fluxo Elétrico — Sistemas de Proteção Térmica
    Significado antes da álgebra: No EX-05, o campo radial \((x,y,z)\) era irrotacional porque cada componente dependia apenas de "sua" variável. Agora vamos ver um caso mais sutil: um campo onde as derivadas cruzadas existem, mas se cancelam perfeitamente por simetria.

    Nos exercícios anteriores, vimos campos irrotacionais onde cada componente dependia apenas de sua própria variável. Agora considere o campo \(\vec{F}(x,y,z) = (yz, xz, xy)\): aqui as componentes dependem de outras variáveis. Será que o rotacional ainda é zero?

    (a) Identifique \(P = yz\), \(Q = xz\), \(R = xy\). Observe a simetria: cada componente é o produto das outras duas variáveis. Calcule a componente \(i\) do rotacional: \(\frac{\partial (xy)}{\partial y} - \frac{\partial (xz)}{\partial z}\).

    (b) Calcule a componente \(j\): \(\frac{\partial (yz)}{\partial z} - \frac{\partial (xy)}{\partial x}\).

    (c) Calcule a componente \(k\): \(\frac{\partial (xz)}{\partial x} - \frac{\partial (yz)}{\partial y}\).

    (d) O rotacional é zero. Diferente do EX-05 (onde cada componente dependia de uma variável só), aqui as derivadas cruzadas são não-zero — mas se cancelam. Explique como a simetria do campo causa esse cancelamento.

    ARQUIVO AUXILIAR [SIGMA-8]
    [NOTA TÉCNICA]: Para a componente \(i\): \(\frac{\partial (xy)}{\partial y} = x\) e \(\frac{\partial (xz)}{\partial z} = x\). A diferença \(x - x = 0\). O mesmo padrão se repete nas outras componentes.
    RELATÓRIO DE CÁLCULO [NÍVEL 4 REQUERIDO]

    SOLUÇÃO:

    (a) \(\frac{\partial (xy)}{\partial y} = x\), \(\frac{\partial (xz)}{\partial z} = x\). Componente \(i\): \(x - x = 0\).

    (b) \(\frac{\partial (yz)}{\partial z} = y\), \(\frac{\partial (xy)}{\partial x} = y\). Componente \(j\): \(y - y = 0\).

    (c) \(\frac{\partial (xz)}{\partial x} = z\), \(\frac{\partial (yz)}{\partial y} = z\). Componente \(k\): \(z - z = 0\).

    (d) \(\nabla \times \vec{F} = (0, 0, 0)\). O campo é irrotacional. A simetria "cada componente é o produto das outras duas variáveis" garante que as derivadas parciais cruzadas são iguais e se cancelam. Este é um caso onde a estrutura do campo, e não a simplicidade de suas dependências, é responsável pela ausência de rotação.
