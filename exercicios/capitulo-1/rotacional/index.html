RELATÓRIO DE ANÁLISE: PROJETO APOLLO-11

Durante a alunissagem, nossa equipe de engenharia identificou padrões inesperados nos campos aerodinâmicos, sugerindo efeitos não triviais do rotacional do campo atmosférico residual na Lua. O presente relatório detalha os cálculos necessários para avaliar o impacto dessas forças.
PRÉ-REQUISITOS

Para resolver estes exercícios, você deve estar confortável com os seguintes conceitos dos tópicos anteriores:

  • Derivadas parciais (Cap 1 — Derivadas Parciais): \(\frac{\partial f}{\partial x}\) como derivada de \(f\) em relação a \(x\), tratando as outras variáveis como constantes. O rotacional é construído combinando derivadas parciais das componentes do campo.
  • Campos vetoriais (Cap 1 — Campo Vetorial): um campo \(\vec{F} = P\hat{i} + Q\hat{j} + R\hat{k}\) associa um vetor a cada ponto. O rotacional mede a tendência de rotação local.
  • Regra do quociente (Cap 0 — Derivadas): para derivar frações como \(\frac{-y}{x^2+y^2}\), usamos \(\frac{d}{dx}\left(\frac{f}{g}\right) = \frac{f\'g - fg\'}{g^2}\).
  • Produto vetorial (Cap 0 — Geometria Analítica): o rotacional pode ser escrito usando o determinante formal, com a mesma estrutura do produto vetorial \(\vec{a} \times \vec{b}\).
  • Regra da cadeia (Cap 0 — Derivadas): quando uma componente depende de \(r = \sqrt{x^2+y^2+z^2}\), precisamos derivar compostas.

    • Resumo: o rotacional \(\nabla \times \vec{F}\) é um vetor cujas componentes são combinações de derivadas parciais. Ele mede a tendência de rotação do campo — como uma pá de moinho giraria na corrente.
