RELATÓRIO DE ANÁLISE: PROJETO APOLLO-11

Durante a alunissagem, nossa equipe de engenharia identificou padrões inesperados nos campos aerodinâmicos, sugerindo efeitos não triviais do rotacional do campo atmosférico residual na Lua. O presente relatório detalha os cálculos necessários para avaliar o impacto dessas forças.
PRÉ-REQUISITOS

Para resolver estes exercícios, você deve estar confortável com os seguintes conceitos dos tópicos anteriores:

  • Derivadas parciais (Cap 1 — Derivadas Parciais): \(\frac{\partial f}{\partial x}\) como derivada de \(f\) em relação a \(x\), tratando as outras variáveis como constantes. O rotacional é construído combinando derivadas parciais das componentes do campo: por exemplo, a componente \(k\) do rotacional envolve \(\frac{\partial Q}{\partial x}\) e \(\frac{\partial P}{\partial y}\).
  • Campos vetoriais (Cap 1 — Campo Vetorial): um campo \(\vec{F}(x,y,z) = P\hat{i} + Q\hat{j} + R\hat{k}\) associa um vetor a cada ponto do espaço. O rotacional é uma propriedade desse campo — mede a tendência de rotação local.
  • Regra do quociente (Cap 0 — Derivadas): para derivar frações como \(\frac{-y}{x^2+y^2}\), usamos \(\frac{d}{dx}\left(\frac{f}{g}\right) = \frac{f'g - fg'}{g^2}\). Essa técnica aparece nos exercícios de nível GAMMA.
  • Produto vetorial (Cap 0 — Geometria Analítica): o rotacional pode ser escrito como o "produto vetorial" do operador \(\nabla\) com o campo \(\vec{F}\), usando o determinante formal. Se você sabe calcular \(\vec{a} \times \vec{b}\), a estrutura é a mesma.
  • Regra da cadeia (Cap 0 — Derivadas): quando uma componente do campo depende de \(r = \sqrt{x^2+y^2+z^2}\), precisamos derivar compostas — por exemplo, \(\frac{\partial}{\partial y}(r^{-3}) = -3r^{-4} \cdot \frac{y}{r}\).

Resumo rápido: o rotacional \(\nabla \times \vec{F}\) é um vetor cujas componentes são combinações de derivadas parciais das componentes de \(\vec{F}\). Ele mede a tendência de rotação do campo em cada ponto — se você colocar uma pequena pá de moinho na corrente, o rotacional diz como ela giraria.
