    ORIGEM: Domínio Definido por Funções - Setor BETA-05

    A parábola era a curva mais simples. Agora o logaritmo entra em cena — uma função cuja inversa é a exponencial. Se você soube inverter \\(y = \\sqrt{x}\\) para \\(x = y^2\\), o mesmo raciocínio se aplica aqui: \\(y = \\ln x\\) se inverte em \\(x = e^y\\). A mecânica da troca de limites é idêntica.

    Considerar \\(D = \\{(x,y) \\mid 1 \\le x \\le 3,\\; 0 \\le y \\le \\ln(x)\\}\\). Calcular \\(\\iint_D \\bigl(2 + e^y\\bigr)\\, dA\\), avaliando primeiro em \\(y\\) e depois invertendo a ordem para checar a consistência.

    (a) Esboce a região \\(D\\) e identifique as curvas \\(y = \\ln x\\), \\(x = 1\\) e \\(x = 3\\).

    (b) Resolva a integral na ordem \\(dy\\,dx\\).

    (c) Reescreva a integral na ordem \\(dx\\,dy\\) e verifique que o resultado é o mesmo.

    ARQUIVO AUXILIAR [SIGMA-2]
    [NOTA TÉCNICA]: Para inverter os limites, note que \\(y = \\ln x\\) equivale a \\(x = e^y\\). Quando \\(x = 1\\), \\(y = 0\\); quando \\(x = 3\\), \\(y = \\ln 3\\).
    RELATÓRIO DE CÁLCULO [SIGMA-4 REQUERIDO]

    (a) A região \\(D\\) fica entre \\(x = 1\\) e \\(x = 3\\), limitada abaixo pelo eixo \\(x\\) e acima pela curva \\(y = \\ln x\\). O ponto mais alto é \\((3, \\ln 3)\\).

    (b) Integração em \\(y\\) primeiro:

    \\( \\int_{x=1}^{3} \\left[\\int_{y=0}^{\\ln(x)} (2 + e^y)\\, dy\\right] dx. \\)

    \\( = \\int_{1}^{3} \\left[ 2y + e^y \\right]_{0}^{\\ln(x)} dx = \\int_{1}^{3} \\left[ 2\\ln(x) + x - 1 \\right] dx. \\)

    \\[ \\int_1^3 (2\\ln x + x - 1)\\, dx = 2\\left[x\\ln x - x\\right]_1^3 + \\left[\\frac{x^2}{2} - x\\right]_1^3 = 2(3\\ln 3 - 3 + 1) + \\left(\\frac{9}{2} - 3 - \\frac{1}{2} + 1\\right) = 6\\ln 3 - 4 + 2 = 6\\ln 3 - 2. \\]

    (c) Ordem \\(dx\\,dy\\): \\(y\\) vai de \\(0\\) a \\(\\ln 3\\), \\(x\\) de \\(e^y\\) a \\(3\\).

    \\[ \\int_0^{\\ln 3} \\int_{e^y}^3 (2 + e^y)\\, dx\\, dy = \\int_0^{\\ln 3} (2 + e^y)(3 - e^y)\\, dy = \\int_0^{\\ln 3} (6 + 3e^y - 2e^y - e^{2y})\\, dy = \\int_0^{\\ln 3} (6 + e^y - e^{2y})\\, dy. \\]

    \\[ = \\left[6y + e^y - \\frac{e^{2y}}{2}\\right]_0^{\\ln 3} = 6\\ln 3 + 3 - \\frac{9}{2} - (0 + 1 - \\frac{1}{2}) = 6\\ln 3 + 3 - 4.5 - 0.5 = 6\\ln 3 - 2. \\]

    Ambos os métodos convergem para \\(6\\ln 3 - 2\\), confirmando Fubini.
