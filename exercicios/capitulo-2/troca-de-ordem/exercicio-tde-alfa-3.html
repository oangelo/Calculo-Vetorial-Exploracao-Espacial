• EXERCÍCIO TDE-ALFA-3
    ORIGEM: Análise de Regiões Tipo I e II - BASE DE TREINAMENTO [1961]

    Considere a integral iterada: \[\int_0^3 \int_1^4 f(x,y)\,dy\,dx\]

    Solicitação:
    • (a) Identifique a região de integração \(R\) e esboce-a no plano \(xy\).
    • (b) Reescreva a integral trocando a ordem de integração para \(dx\,dy\).
    • (c) Classifique a região \(R\) como Tipo I, Tipo II, ou ambos. Justifique.

    DEFINIÇÕES:
    Região Tipo I: \(R = \{(x,y) : a \leq x \leq b,\; g_1(x) \leq y \leq g_2(x)\}\)
    Região Tipo II: \(R = \{(x,y) : c \leq y \leq d,\; h_1(y) \leq x \leq h_2(y)\}\)

    ARQUIVO AUXILIAR [SIGMA-1]
    [NOTA TÉCNICA]: Quando os limites são todos constantes, a região é um retângulo. Um retângulo pode ser descrito tanto como Tipo I quanto como Tipo II.
    RELATÓRIO DE CÁLCULO [SIGMA-1 REQUERIDO]

    SOLUÇÃO:

    (a) A região de integração é o retângulo: \[R = \{(x,y) : 0 \leq x \leq 3,\; 1 \leq y \leq 4\} = [0, 3] \times [1, 4]\] No plano \(xy\), é um retângulo com vértices em \((0,1)\), \((3,1)\), \((3,4)\) e \((0,4)\).

    (b) Trocando a ordem de integração: \[\int_1^4 \int_0^3 f(x,y)\,dx\,dy\]

    (c) A região é ambos (Tipo I e Tipo II):

    • Como Tipo I: \(a=0\), \(b=3\), \(g_1(x)=1\), \(g_2(x)=4\) (funções constantes)
    • Como Tipo II: \(c=1\), \(d=4\), \(h_1(y)=0\), \(h_2(y)=3\) (funções constantes)

    Regiões retangulares são o caso mais simples onde a classificação dupla é possível, facilitando a troca de ordem de integração.
