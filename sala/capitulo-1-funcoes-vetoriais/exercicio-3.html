Quando o rotacional de um campo é zero em todo ponto, dizemos que o campo é irrotacional. Se esse campo também é o gradiente de alguma função escalar, ele é chamado de campo conservativo — ou seja, o trabalho realizado ao longo de qualquer caminho fechado é zero. Isso aparece em Física 3: o campo elétrico eletrostático é conservativo (por isso existe o potencial elétrico), mas o campo magnético não é.

b) O rotacional de $\vec{G}$ é (zero / não zero) $\Rightarrow$ $\vec{G}$ descreve (ex: campo sem vórtice / campo com rotação)

c) Além disso, $\vec{F} = \nabla f$, então $\vec{F}$ é (conservativo / não conservativo)
